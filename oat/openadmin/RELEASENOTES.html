Description

The IBM® OpenAdmin Tool (OAT) for Informix is a web application for administering and analyzing the performance of IBM Informix database servers. You can administer multiple database server instances from a single OAT installation on a web server. You can access the web server through any browser to administer all your database servers.

OAT includes these plug-ins: the IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT), the IBM Informix Replication Plug-in for OpenAdmin Tool (OAT), the IBM Informix Schema Manager Plug-in for OpenAdmin Tool (OAT), and the IBM Informix TimeSeries Plug-in for OpenAdmin Tool (OAT). You can create additional plug-ins for OAT to add the functions that you need.

For a complete list of what's new in this release, go to http://pic.dhe.ibm.com/infocenter/informix/v121/topic/com.ibm.po.doc/new_features_ce.htm.

Installing IBM OpenAdmin Tool (OAT) for Informix 3.11

OAT is installed by default during a typical installation of the following products:

  • IBM Informix Client Software Development Kit (Client SDK), Version 4.10
  • IBM Informix Connect, Version 4.10

For installation instructions, see the topic Installing the OpenAdmin Tool for Informix with the Client SDK in the information center: http://pic.dhe.ibm.com/infocenter/informix/v121/topic/com.ibm.cpi.doc/ids_cpi_027.htm

If you are upgrading from an earlier OAT version, additional information is in the topic Upgrading from a previous version of the OpenAdmin Tool (OAT) for Informix: http://pic.dhe.ibm.com/infocenter/informix/v121/topic/com.ibm.cpi.doc/ids_cpi_031.htm

Alternatively, the OpenAdmin Tool Community Edition is available from the IBM Informix Free Product Download page: https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?lang=en_US&source=swg-informixfpd . The OpenAdmin Tool Community Edition includes OAT and the optional plug-ins, but it does not include the supporting software. This edition must be installed manually. Refer to the README file for installation instructions.
