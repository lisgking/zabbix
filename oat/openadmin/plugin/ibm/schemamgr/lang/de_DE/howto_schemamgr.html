Informix-Plug-in 'Schema Manager' für OAT

Das IBM® Informix®-Plug-in 'Schema Manager' für OpenAdmin Tool (OAT) stellt eine grafische Oberfläche zum Anzeigen und Verwalten von Datenbanken und Tabellen bereit.

Mit dem Plug-in 'Schema Manager' können Sie die folgenden Tasks ausführen:

  • Erstellen und Löschen von Datenbanken, Tabellen und Indizes.
  • Laden und entladen Sie außerhalb des Datenbankservers gespeicherte Daten mithilfe von externen Tabellen.
  • Überwachen von Informationen zu den Tabellen- und Fragmentstatistikdaten.
  • Angeben, ob die Datenverteilungsstatistikdaten nur anhand der Fragmente berechnet werden, die sich in einer fragmentierten Tabelle ändern, oder anhand der gesamten Tabelle.
  • Anzeigen von Informationen zu Datenbankobjekten einschließlich gespeicherter Prozeduren und Funktionen, Sequenzen, benutzerdefinierter Datentypen, Zugriffsrechten, DataBlade®-Modulen, Ergebnistabellen, Umsetzungen und Operatorklassen.
  • Anzeigen von Informationen zu Tabellenobjekten einschließlich Spalten, Indizes, Verweisen, Zugriffsrechten, benutzerdefinierter Datentypen, Integritätsbedingungen, Fragmenten und Triggern.

Anzeigen von Datenbankobjekten

Zeigen Sie Informationen zu gespeicherten Prozeduren und Funktionen, Sequenzen, benutzerdefinierten Datentypen, Zugriffsrechten, DataBlade-Modulen, Ergebnistabellen, Umsetzungen und Operatorklassen an.

Gehen Sie wie folgt vor, um Datenbankobjekte anzuzeigen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbank in der Liste aus.

    Auf der Seite wird ein Fenster mit Informationen zu jedem der Datenbankobjekte angezeigt.

  3. Klicken Sie im Objektfenster auf die Erweiterungsschaltfläche, um weitere Details zu einem Datenbankobjekt anzuzeigen.

Anzeigen von Tabellenobjekten

Zeigen Sie Informationen zu Spalten, Indizes, Verweisen, Zugriffsrechten, benutzerdefinierten Datentypen, Integritätsbedingungen, Fragmenten und Triggern an.

Gehen Sie wie folgt vor, um Tabellenobjekte anzuzeigen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Wenn Sie eine Tabelle suchen wollen, geben Sie einen beliebigen Teil des Tabellennamens in das Feld Suchen ein und klicken auf Suchen. Sie können auch eine Datenbank erweitern, um die Tabellen anzuzeigen.
  3. Optional: Wählen Sie die Filter aus, um mehr Tabellen in die Liste aufzunehmen:
    • Wählen Sie Kataloge einschließen aus, um Systemkataloge aufzunehmen.
    • Wählen Sie Sichten einschließen aus, um Sichten aufzunehmen. Eine Sicht ist eine virtuelle Tabelle, die auf einer angegebenen Anweisung SELECT basiert.
    • Wählen Sie Synonyme einschließen aus, um Synonyme aufzunehmen. Ein Synonym ist ein alternativer Name für eine Tabelle, eine Sicht oder ein Sequenzobjekt. Normalerweise verweisen Synonyme auf Tabellen, die sich nicht in der aktuellen Datenbank befinden.
  4. Wählen Sie die Tabelle in der Liste aus.

    Auf der Seite wird ein Fenster mit Informationen zu jedem der Tabellenobjekte angezeigt.

  5. Klicken Sie im Objektfenster auf die Erweiterungsschaltfläche, um weitere Details zu einem Tabellenobjekt anzuzeigen.

Überprüfen von Tabellenstatistikdaten

Überprüfen Sie Informationen zu Tabellen- und Fragmentstatistikdaten einschließlich des Zeitpunkts der letzten Erfassung der Statistikdaten, der Datenverteilung und der Prozentzahl der geänderten Zeilen. Geben Sie die Optionen für die automatische Aktualisierung von Tabellenstatistikdaten an.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix Version 11.70, um die Optionen für die automatische Aktualisierung von Tabellenstatistikdaten anzugeben und um einige der Informationen, z.B. die Prozentzahl der geänderten Zeilen, anzuzeigen.

Gehen Sie wie folgt vor, um Tabellenstatistikdaten zu überprüfen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie die Datenbank in der Liste.
  3. Wählen Sie die Tabelle in der Liste aus.
  4. Erweitern Sie die Anzeige Statistik.
  5. Wenn Sie die fragmentbezogenen Statistikdaten anzeigen wollen, klicken Sie auf Fragmentstatistik. Diese Informationen werden für IBM Informix Version 11.70 angezeigt.

Erstellen einer Datenbank

Erstellen Sie eine Datenbank und geben Sie die Ländereinstellung für die Datenbank an.

Voraussetzung:
OpenAdmin Tool (OAT) for Informix erfordert IBM Informix 11.70.xC2, um eine Datenbank zu erstellen.


Gehen Sie wie folgt vor, um eine Datenbank zu erstellen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Klicken Sie im Menü Aktionen auf Datenbank erstellen.
  3. Führen Sie die Anweisungen im Assistenten Datenbank erstellen aus.

  • Zugehörige Konzepte:
  • Verwenden von CREATE DATABASE (Konzepte und Implementierung von Datenbanken)
  • Datenbankländereinstellung (GLS Benutzerhandbuch)
  • Zugehörige Referenz:
  • CREATE DATABASE statement (SQL Syntax)
  • create database argument: Create a database (SQL administration API) (Administrator's Reference)

Löschen einer Datenbank

Entfernen Sie eine Datenbank und löschen Sie die darin gespeicherten Tabellen und Daten.

Voraussetzung:
OpenAdmin Tool (OAT) for Informix erfordert IBM Informix 11.70.xC2, um eine Datenbank zu löschen.


Gehen Sie wie folgt vor, um eine Datenbank zu löschen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbank aus.
  3. Klicken Sie im Menü Aktionen auf Datenbank löschen.

  • Zugehörige Referenz:
  • DROP DATABASE statement (SQL Syntax)

Verwalten von Tabellen und Indizes

Erstellen und löschen Sie Tabellen und schneiden Sie Tabellen ab. Erstellen, inaktivieren und löschen Sie Indizes.

Erstellen einer Tabelle

Erstellen Sie eine Tabelle und definieren Sie die Attribute, um eine Tabelle mit vollem Funktionsumfang auszustatten.

Gehen Sie wie folgt vor, um eine Tabelle zu erstellen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbank aus, in der die Tabelle erstellt werden soll.
  3. Klicken Sie im Menü Aktionen auf Tabelle erstellen.
  4. Führen Sie die Anweisungen im Assistenten Tabelle erstellen aus.

Löschen einer Tabelle

Entfernen Sie eine Tabelle mit ihren zugehörigen Indizes und Daten.

Wenn Sie eine Tabelle entfernen, löschen Sie auch die in ihr gespeicherten Daten sowie alle Indizes, Integritätsbedingungen, Trigger, Zugriffsberechtigungen und lokalen Synonyme. Darüber hinaus entfernen Sie alle auf der Tabelle basierenden Sichten sowie alle zugehörigen Übertretungs- und Diagnosetabellen.

Gehen Sie wie folgt vor, um eine Tabelle zu löschen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie in der Liste die Datenbank, die die Tabelle enthält.
  3. Wählen Sie die Tabelle aus.
  4. Klicken Sie im Menü Aktionen auf Tabelle löschen.

Abschneiden einer Tabelle

Löschen Sie alle Zeilen aus der Tabelle und geben Sie den zugehörigen Speicherbereich frei oder reservieren Sie den Speicherbereich zwecks Wiederverwendung durch die Tabelle und ihre Indizes.

Die Wiederverwendung des Speicherbereichs kann die Speicherverwaltung in Anwendungen effizienter machen, in denen eine Tabelle regelmäßig geleert und mit neuen Zeilen erneut geladen wird.

Gehen Sie wie folgt vor, um eine Tabelle abzuschneiden:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie in der Liste die Datenbank, die die Tabelle enthält.
  3. Wählen Sie die Tabelle aus.
  4. Klicken Sie im Menü Aktionen auf Tabelle abschneiden.
  5. Optional: Wählen Sie Speicherbereich reservieren aus, um den Speicherbereich beizubehalten, der der Tabelle zugeordnet ist.

Erstellen eines Index

Erstellen Sie einen Index für mindestens eine Spalte in einer Tabelle.

Gehen Sie wie folgt vor, um einen Index zu erstellen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie in der Liste die Datenbank und wählen Sie die Tabelle aus.
  3. Klicken Sie im Menü Aktionen auf Index erstellen.
  4. Befolgen Sie die Anweisungen im Assistenten zum Erstellen eines Index.

  • Zugehörige Konzepte:
  • Aspekte von Indizes und der Indexleistung (Optimierung)
  • Zugehörige Referenz:
  • CREATE INDEX statement (SQL Syntax)

Inaktivieren oder Aktivieren eines Index

Inaktivieren Sie einen Index oder aktivieren Sie einen zuvor inaktivierten Index.

Ein Index ist standardmäßig aktiviert. Der Datenbankserver erkennt den Index und aktualisiert ihn während Einfüge-, Lösch- und Aktualisierungsoperationen. Ist ein Index inaktiviert, wird er bei diesen Operationen nicht mehr aktualisiert.

Gehen Sie wie folgt vor, um einen Index zu inaktivieren:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie in der Liste die Datenbank mit der Tabelle, für die der Index gilt, und wählen Sie die Tabelle aus.
  3. Erweitern Sie die Anzeige Indizes.
  4. Wählen Sie den Index aus.
  5. Klicken Sie im Menü Aktionen für die Anzeige auf Index aktivieren oder auf Index inaktivieren.

  • Zugehörige Konzepte:
  • How the Database Server Treats Disabled Indexes (SQL Syntax)
  • Definitions of Database Object Modes (SQL Syntax)

Löschen eines Index

Löschen Sie einen Index, beispielsweise weil er nicht mehr gültig ist.

Gehen Sie wie folgt vor, um einen Index zu löschen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Erweitern Sie in der Liste die Datenbank mit der Tabelle, für die der Index gilt, und wählen Sie die Tabelle aus.
  3. Erweitern Sie die Anzeige Indizes.
  4. Wählen Sie den Index aus.
  5. Klicken Sie im Menü Aktionen für die Anzeige auf Index löschen.

  • Zugehörige Tasks:
  • Löschen von Indizes für effizientere Tabellenaktualisierung (Optimierung)
  • Zugehörige Referenz:
  • DROP INDEX statement (SQL Syntax)

Laden und Entladen von Daten mit externen Tabellen

Laden und entladen Sie außerhalb des Datenbankservers gespeicherte Daten mithilfe von externen Tabellen.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Laden von Daten mit externer Tabelle

Laden Sie Daten aus einer Quelle, die sich außerhalb des Datenservers befindet, mit einer externen Tabelle in eine Datenbanktabelle.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Gehen Sie wie folgt vor, um Daten mit einer externen Tabelle zu laden:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbanktabelle aus, in die die Daten geladen werden sollen.

    Wenn Sie eine Tabelle suchen wollen, geben Sie einen beliebigen Teil des Tabellennamens in das Feld Suchen ein und klicken auf Suchen. Sie können auch eine Datenbank erweitern, um die Tabellen anzuzeigen.

  3. Klicken Sie im Menü Aktionen auf Tabelle laden.
  4. Führen Sie die Anweisungen im Assistenten Tabelle laden aus.

Entladen von Daten in externe Tabelle

Entladen Sie Daten aus einer Datenbanktabelle in eine externe Tabelle.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Gehen Sie wie folgt vor, um Daten in eine externe Tabelle zu entladen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbanktabelle aus, aus der die Daten entladen werden sollen.

    Wenn Sie eine Tabelle suchen wollen, geben Sie einen beliebigen Teil des Tabellennamens in das Feld Suchen ein und klicken auf Suchen. Sie können auch eine Datenbank erweitern, um die Tabellen anzuzeigen.

  3. Klicken Sie im Menü Aktionen auf Tabelle entladen.
  4. Führen Sie die Anweisungen im Assistenten Tabelle entladen aus.

Erstellen einer externen Tabelle

Erstellen Sie eine Tabelle, die außerhalb des Datenbankservers verwaltet und gespeichert wird, um Daten zu laden und zu entladen.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um externe Tabellen zu erstellen.

Gehen Sie wie folgt vor, um eine externe Tabelle zu erstellen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Wählen Sie die Datenbank aus, für die die Tabelle erstellt werden soll.
  3. Klicken Sie im Menü Aktionen auf Tabelle erstellen.
  4. Führen Sie die Anweisungen im Assistenten Tabelle erstellen aus.

    Wählen Sie für Tabellentyp einen Typ einer externen Tabelle aus:

    • Extern mit fester Satzlänge: Eine Tabelle im Textformat mit fester Breite, deren Daten außerhalb des Informix-Datenbankservers verwaltet und gespeichert werden.
    • Extern mit Begrenzer: Eine Tabelle im Textformat mit Trennzeichen, deren Daten außerhalb des Informix-Datenbankservers verwaltet und gespeichert werden.
    • Extern Informix: Eine Tabelle im internen Informix-Format, deren Daten außerhalb des Informix-Datenbankservers verwaltet und gespeichert werden.

      Externe Tabellen werden verwendet, um Daten für Ihre Datenbank zu laden und zu entladen.

Gehen Sie wie folgt vor, um Informationen zu einer externen Tabelle anzuzeigen:

  1. Klicken Sie im Menü für IBM OpenAdmin Tool (OAT) for Informix auf SQL Toolbox > Schemamanager.
  2. Wenn Sie eine Tabelle suchen wollen, geben Sie einen beliebigen Teil des Tabellennamens in das Feld Suchen ein und klicken auf Suchen. Sie können auch eine Datenbank erweitern, um die Tabellen anzuzeigen.
  3. Wählen Sie die Tabelle in der Liste aus.

    Auf der Seite wird ein Fenster mit Informationen zu jedem der Tabellenobjekte angezeigt.

    Die Anzeige Tabelleninformationen und die Anzeige Spalten zeigen spezifische Informationen zu externen Tabellen an.

Anzeigen des Status von Lade- und Entladetask

Zeigen Sie den Status der von IBM OpenAdmin Tool (OAT) for Informix übergebenen Tasks an, um Daten mit externen Tabellen zu laden und zu entladen.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Gehen Sie wie folgt vor, um den Status von Lade- und Entladetasks anzuzeigen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Klicken Sie auf die Datenbank oder die Tabelle, für die die Task ausgeführt wird.
  3. Erweitern Sie die Anzeige Informationen.
  4. Klicken Sie auf die Registerkarte Taskstatus.

    Wenn Sie weitere Informationen zu einer Task, einschließlich der Fehlercodes und Antwortnachrichten, anzeigen wollen, klicken Sie auf das Detailsymbol der Task.

  • Zugehörige Konzepte:
  • Error Messages (Administrator's Reference)

Ausführen gespeicherter Lade- und Entladetasks

Führen Sie Tasks aus, die Daten mithilfe externer Tabellen laden und entladen.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Wenn Sie Daten mithilfe von externen Tabellen laden oder entladen, wird die Task gespeichert. Sie können die Task erneut ausführen oder löschen.

Gehen Sie wie folgt vor, um eine gespeicherte Lade- oder Entladetasks auszuführen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Klicken Sie auf die Datenbank oder die Tabelle, für die die Task ausgeführt wird.
  3. Erweitern Sie die Anzeige Informationen.
  4. Klicken Sie auf die Registerkarte Tasks.
  5. Wählen Sie die auszuführenden Tasks aus.
  6. Klicken Sie im Menü Aktionen für die Anzeige auf Übergeben.

Löschen von Lade- und Entladetasks

Löschen Sie eine Lade- oder Entladetask oder den Status einer Lade- oder Entladetask.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Wenn Sie Daten mithilfe von OAT laden oder entladen, wird die Task gespeichert. Sie können die Task erneut übergeben oder löschen. Sie können den Status der übergebenen Tasks auch löschen.

Wichtig:
Wenn Sie auf der Seite Task eine Task löschen, werden die Ergebnisse der Task auch auf der Seite Taskstatus gelöscht. Wenn Sie einen Taskstatus löschen, wird nur der Taskstatus gelöscht.

Gehen Sie wie folgt vor, um Lade- und Entladetasks zu löschen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Klicken Sie auf die Datenbank oder die Tabelle, für die die Task ausgeführt wird.
  3. Erweitern Sie die Anzeige Informationen.
  4. Wählen Sie die entsprechende Registerkarte aus:
    • Wenn Sie die Task und den Taskstatus löschen wollen, klicken Sie auf die Registerkarte Tasks.
    • Wenn Sie nur den Taskstatus löschen wollen, klicken Sie auf die Registerkarte Taskstatus.
  5. Wählen Sie die Task aus.
  6. Klicken Sie auf Aktionen > Löschen.

Erstellen einer Gruppe gespeicherter Lade- und Entladetasks

Gruppieren Sie gespeicherte Tasks so, dass diese gemeinsam übergeben werden können.

Voraussetzung:
IBM OpenAdmin Tool (OAT) for Informix erfordert IBM Informix ab Version 11.50.xC6, um Daten mit externen Tabellen zu laden und zu entladen.

Wenn Sie Daten mithilfe von OAT laden oder entladen, wird die Task gespeichert. Sie können die Task erneut ausführen, löschen oder mit anderen gespeicherten Tasks in einer Gruppe zusammenfassen.

Gehen Sie wie folgt vor, um eine Gruppe gespeicherter Lade- oder Entladetasks zu erstellen:

  1. Klicken Sie im OAT-Menü auf SQL Toolbox > Schemamanager.
  2. Klicken Sie auf die Datenbank oder die Tabelle, für die die Task ausgeführt wird.
  3. Erweitern Sie die Anzeige Informationen.
  4. Klicken Sie auf die Registerkarte Tasks.
  5. Wählen Sie die Tasks aus, die zu einer Gruppe gehören sollen. Lade- und Entladetasks können in derselben Gruppe zusammengefasst werden.
  6. Klicken Sie im Menü Aktionen für die Anzeige auf Gruppe erstellen und geben Sie einen Namen für die Gruppe ein.
