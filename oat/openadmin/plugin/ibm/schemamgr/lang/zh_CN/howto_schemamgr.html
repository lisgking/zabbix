Informix Schema Manager Plug-in for OAT

IBM® Informix® Schema Manager Plug-in for OpenAdmin Tool (OAT) 提供图形界面，用于查看和管理数据库和表。

通过该模式管理器插件，您可以执行以下任务：

  • 创建并删除数据库、表和索引。
  • 使用外部表装入和卸载在数据库服务器外部管理和存储的数据。
  • 监视有关表和分段统计信息的信息。
  • 指定仅对已在分段表中更改的分段还是对整张表计算数据分发统计信息。
  • 查看有关数据库对象的信息，包括存储过程和函数、序列、用户定义的数据类型 (UDT)、特权、DataBlade® 模块、聚集、强制转型和运算符类。
  • 查看关于表对象的信息，包括列、索引、引用、特权、UDT、约束、分段和触发器。

查看数据库对象

查看有关存储过程和函数、序列、用户定义的数据类型 (UDT)、特权、DataBlade 模块、聚集、强制转型和运算符类的信息。

要查看数据库对象：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中选择数据库。

    页面显示一个面板，以提供有关每个数据库对象的信息。

  3. 要显示有关数据库对象的更多详细信息，请单击对象面板上的展开按钮。

查看表对象

查看有关列、索引、引用、特权、用户定义的数据类型 (UDT)、约束、分段和触发器的信息。

要查看表对象：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 要查找表，请在查找字段中输入表名称的任何部分，然后单击搜索。也可以展开数据库来显示各表。
  3. 可选：选择过滤器，以在列表中包含更多表：
    • 要包含系统目录，请选择包含目录。
    • 要包含视图，请选择包含视图。视图是基于特定 SELECT 语句的一个虚拟表。
    • 要包含同义词，请选择包含同义词。同义词是表、视图或序列对象的替代名称。通常，同义词是指不在当前数据库中的表。
  4. 在列表中选择表。

    页面显示一个面板，以提供有关每个表对象的信息。

  5. 要显示有关表对象的更多详细信息，请单击对象面板上的展开按钮。

查看表统计信息

查看有关表和分段统计信息的信息，包括上次记录统计信息的时间、数据分发和已更改行的百分比。指定自动更新表统计信息的选项。

先决条件：
IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.70 来指定自动更新表统计信息的选项，以及显示一些信息，例如已更改行的百分比。

要查看表统计信息：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中展开数据库。
  3. 在列表中选择表。
  4. 展开统计信息面板。
  5. 要显示分段级别的统计信息，单击分段统计信息。将对 IBM Informix V11.70 显示此信息。

创建数据库

创建数据库并指定数据库语言环境。

先决条件：
要创建数据库，OpenAdmin Tool (OAT) for Informix 需要 IBM Informix 11.70.xC2。


要创建数据库：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在操作菜单上，单击创建数据库。
  3. 遵循“创建数据库”向导中的指导信息。

  • 相关概念：
  • 使用 CREATE DATABASE（数据库设计指南）
  • 数据库语言环境（GLS 用户指南）
  • 相关参考：
  • CREATE DATABASE 语句（SQL 语法）
  • 创建数据库参数：创建数据库（SQL 管理 API）（管理员参考）

删除数据库

除去数据库并删除其中存储的表和数据。

先决条件：
要删除数据库，OpenAdmin Tool (OAT) for Informix 需要 IBM Informix 11.70.xC2。


要删除数据库：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 选择该数据库。
  3. 在操作菜单上，单击删除数据库。

  • 相关参考：
  • DROP DATABASE 语句（SQL 语法）

管理表和索引

创建、删除和截断表。创建、禁用和删除索引。

创建表

创建表并定义使该表能够执行全部功能所必需的属性。

要创建表：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 选择要在其中创建表的数据库。
  3. 在操作菜单上，单击创建表。
  4. 遵循“创建表”向导中的指导信息。

删除表

除去表及与之关联的索引和数据。

除去表时，同时会删除表中存储的数据以及所有索引、约束、触发器、访问特权和本地同义词。另外，还会除去基于表的所有视图以及与表关联的所有违例和诊断表。

要删除表：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中，展开包含表的数据库。
  3. 选择表。
  4. 在操作菜单上，单击删除表。

截断表

删除表中的所有行并释放关联的存储空间，或者保留存储空间以供表及其索引复用。

在定期清空表并重新装入新行的应用程序中，复用存储空间可使其存储管理更加有效。

要截断表：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中，展开包含表的数据库。
  3. 选择表。
  4. 在操作菜单上，单击截断表。
  5. 可选：选择保留存储空间以保留分配给表的存储空间。

创建索引

为表中一个或多个列创建索引。

要创建索引：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中，展开数据库并选择表。
  3. 在操作菜单上，单击创建索引。
  4. 遵循“创建索引”向导中的指导信息。

  • 相关概念：
  • 索引和索引性能注意事项（性能指南）
  • 相关参考：
  • CREATE INDEX 语句（SQL 语法）

启用或禁用索引

禁用索引，或启用已禁用的索引。

缺省情况下，索引是启用的。在进行插入、删除和更新操作期间，数据库服务器将识别该索引并进行更新。如果禁用索引，那么在进行这些操作期间将不再对其进行更新。

要启用或禁用索引：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中，展开包含应用索引的表的数据库，并选择表。
  3. 展开索引面板。
  4. 选择索引。
  5. 在面板的操作菜单上，单击启用索引或禁用索引。

  • 相关概念：
  • 数据库服务器如何处理已禁用的索引（SQL 语法）
  • 数据库对象方式的定义（SQL 语法）

删除索引

删除索引；例如，除去不再有效的索引。

要删除索引：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 在列表中，展开包含应用索引的表的数据库，并选择表。
  3. 展开索引面板。
  4. 选择索引。
  5. 在面板的操作菜单上，单击删除索引。

  • 相关任务：
  • 删除索引以提高表更新效率（性能指南）
  • 相关参考：
  • DROP INDEX 语句（SQL 语法）

使用外部表装入和卸载数据

使用外部表装入和卸载在数据库服务器外部管理和存储的数据。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

装入外部表的数据

通过使用外部表，将数据从数据库服务器的外部源装入数据库表中。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

要装入外部表的数据：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 选择要装入数据的数据库表。

    要查找表，请在查找字段中输入表名称的任何部分，然后单击搜索。也可以展开数据库来显示各表。

  3. 在操作菜单上，单击装入表。
  4. 遵循“装入表”向导中的指导信息。

将数据卸载到外部表

将数据从数据库表卸载到外部表。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

要将数据卸载到外部表：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 选择要从中卸载数据的数据库表。

    要查找表，请在查找字段中输入表名称的任何部分，然后单击搜索。也可以展开数据库来显示各表。

  3. 在操作菜单上，单击卸载表。
  4. 遵循“卸载表”向导中的指导信息。

创建外部表

创建在数据库服务器外部管理和存储的表，以装入和卸载数据。

先决条件：
要创建外部表，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6。

要创建外部表：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 选择要为其创建表的数据库。
  3. 在操作菜单上，单击创建表。
  4. 遵循“创建表”向导中的指导信息。

    对于表类型，请选择外部表类型：

    • 外部固定：固定宽度文本格式的表，其中的数据在 Informix 数据库服务器以外管理和存储。
    • 外部定界：定界文本格式的表，其中的数据在 Informix 数据库服务器以外管理和存储。
    • 外部 Informix：内部 Informix 格式的表，其中的数据在 Informix 数据库服务器以外管理和存储。

      外部表用于为您的数据库装入和卸载数据。

查看外部表信息

查看关于外部表的信息。

将对 Informix 11.50.xC6 或更高版本显示此信息。

要查看外部表的信息：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 要查找表，请在查找字段中输入表名称的任何部分，然后单击搜索。也可以展开数据库来显示各表。
  3. 在列表中选择表。

    页面显示一个面板，以提供有关每个表对象的信息。

  4. 要显示有关表对象的更多详细信息，请单击对象面板上的展开按钮。

    表信息面板和列面板显示特定于外部表的信息。

查看装入和卸载任务的状态

查看通过 IBM OpenAdmin Tool (OAT) for Informix 提交的任务的状态，以使用外部表装入和卸载数据。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

要查看装入和卸载任务的状态：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 单击数据库或在其上运行该任务的表。
  3. 展开信息面板。
  4. 单击任务状态选项卡。

    要显示有关任务的更多信息（包括错误代码和返回消息），请单击任务的详细信息图标。

  • 相关概念：
  • Error Messages (Administrator's Reference)

运行已保存的装入和卸载任务

使用外部表运行装入和卸载数据的任务。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

使用外部表装入或卸载数据时，将保存该任务。您可以再次运行该任务或将其删除。

要运行已保存的装入或卸载任务：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 单击数据库或在其上运行该任务的表。
  3. 展开信息面板。
  4. 单击任务选项卡。
  5. 选择要运行的任务。
  6. 在面板的操作菜单上，单击提交。

删除装入和卸载任务

删除装入或卸载任务，或删除装入或卸载任务的状态。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

使用 OAT 装入或卸载数据时，将保存该任务。您可以再次提交该任务或将其删除。您也可以删除已提交任务的状态。

重要信息：
当删除任务页面上的任务时，同时会将任务的结果从任务状态页面删除。当删除任务状态时，仅会删除该任务状态。

要删除装入和卸载任务：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 单击数据库或在其上运行该任务的表。
  3. 展开信息面板。
  4. 选择适当的选项卡：
    • 要删除任务和任务状态，请单击任务选项卡。
    • 要仅删除任务状态，请单击任务状态选项卡。
  5. 选择该任务。
  6. 单击操作 > 删除。

创建已保存的装入或卸载任务组

将已保存的任务集结成组以便一起提交。

先决条件：
要使用外部表装入和卸载数据，IBM OpenAdmin Tool (OAT) for Informix 需要 IBM Informix V11.50.xC6 或更高版本。

使用 OAT 装入或卸载数据时，将保存该任务。您可以再次运行该任务、将其删除或将其与其他已保存的任务分为一组。

要创建已保存的装入或卸载任务组：

  1. 在 OAT 菜单上，单击 SQL 工具箱 > 模式管理器。
  2. 单击数据库或在其上运行该任务的表。
  3. 展开信息面板。
  4. 单击任务选项卡。
  5. 选择要包含在组中的任务。装入和卸载任务可以包含在同一组中。
  6. 在面板的操作菜单上，单击创建组，然后输入组的名称。
