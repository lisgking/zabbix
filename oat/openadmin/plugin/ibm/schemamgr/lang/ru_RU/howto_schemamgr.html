Подключаемый модуль менеджера схем для IBM OpenAdmin Tool for Informix

Подключаемый модуль менеджера схем для IBM® OpenAdmin Tool (OAT) for Informix предоставляет графический интерфейс для просмотра и администрирования баз данных и таблиц.

С помощью подключаемого модуля менеджера схем можно выполнять следующие задачи:

  • Создание таблиц и индексов.
  • Загрузка и выгрузка данных, которые хранятся и обрабатываются во внешних источниках, с использованием внешних таблиц.
  • Просмотр информации об объектах баз данных, включая хранимые процедуры и функции, последовательности, пользовательские типы данных, привилегии, модули DataBlade, агрегированные данные, преобразования и классы операторов.
  • Просмотр информации об объектах таблиц, включая столбцы, индексы, ссылки, привилегии, пользовательские типы данных, ограничения, фрагменты и триггеры.

Просмотр объектов базы данных

Просмотр сведений о хранимых процедурах и функциях, последовательностях, пользовательских типах данных, привилегиях, модулях DataBlade, агрегированных типах данных, преобразованиях и классах операторов.

Для просмотра объектов базы данных выполните следующее:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Выберите базу данных в списке.

    На странице будет показана информационная панель для каждого объекта базы данных.

  3. Для вывода на экран дополнительных сведений об объекте базы данных щелкните кнопку раскрытия на панели объектов.

Просмотр объектов таблиц

Просмотр сведений о столбцах, индексах, ссылках, привилегиях, пользовательских типах данных, ограничениях, фрагментах и триггерах.

Для просмотра объектов таблицы выполните следующее:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Чтобы найти таблицу, начните вводить любую часть ее имени в поле Найти для вывода на экран соответствующих таблиц. Можно также раскрыть базу данных, чтобы вывести список ее таблиц.
  3. Необязательно: Выберите фильтры для включения дополнительных таблиц в список:
    • Чтобы включить в список системные каталоги, выберите пункт Включить каталоги.
    • Чтобы включить в список представления, выберите пункт Включить представления. Представление — это виртуальная таблица, созданная на основе заданного оператора SELECT.
    • Чтобы включить синонимы, выберите пункт Включить синонимы. Синоним - это альтернативное имя таблицы, представления или последовательности. Обычно синонимы относятся к таблицам, которые находятся за пределами текущей базы данных.
  4. Выберите таблицу в списке.

    На странице будет показана панель со сведениями о каждом объекте таблицы.

  5. Для вывода на экран дополнительных сведений об объекте таблицы щелкните кнопку раскрытия на панели объектов.

Просмотр статистики таблицы и фрагмента

Просмотрите информацию о таблицах и фрагментах, включая процент измененных строк и число операций UPDATE, DELETE и INSERT для строк. Просмотрите, когда в последний раз записывалась статистика и сколько времени потребовалось для ее вычисления.

Требуемый компонент:
Для IBM OpenAdmin Tool (OAT) for Informix требуется IBM Informix версии 11.70, чтобы вывести на экран некоторую информацию, включая процент измененных строк, число операций и время построения, которое потребовалось для вычисления статистики.

Чтобы просмотреть статистику таблицы и фрагмента:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Раскройте базу данных в списке.
  3. Выберите таблицу в списке.
  4. Раскройте панель Статистика.
  5. Чтобы показать статистику на уровне фрагмента, щелкните по Статистика фрагмента. Эта информация выводится для IBM Informix версии 11.70.

Управление таблицами и индексами

Создание, отбрасывание и усечение таблиц. Создание, отключение и отбрасывание индексов.

Создание таблицы

Создайте таблицу и определите атрибуты, необходимые, чтобы сделать таблицу полнофункциональной.

Чтобы создать таблицу, выполните следующее:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Выберите базу данных, в которой нужно создать таблицу.
  3. В меню Действия выберите пункт Создать таблицу.
  4. Следуйте инструкциям в мастере создания таблицы.

Полное удаление таблицы

Удалите таблицу со всеми связанными индексами и данными.

При удалении таблицы также удаляются хранимые в ней данные, индексы, ограничения, триггеры, права доступа и локальные синонимы. Также удаляются все ее табличные представления и связанные с ней таблицы нарушений и диагностики.

Чтобы полностью удалить таблицу, выполните следующее:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Раскройте в списке базу данных, содержащую таблицу.
  3. Выберите таблицу.
  4. В меню Действия выберите пункт Полностью удалить таблицу.

Усечение таблицы

Удалите все строки в таблице и освободите связанное с ними пространство хранения или зарезервируйте это пространство для повторного использования таблицей и ее индексами.

Благодаря повторному использованию пространства хранения управление хранением становится более эффективным в программах, в которых таблица периодически очищается и перезагружается с новыми строками.

Чтобы произвести усечение таблицы, выполните следующее:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Раскройте в списке базу данных, содержащую таблицу.
  3. Выберите таблицу.
  4. В меню Действия выберите пункт Произвести усечение таблицы.
  5. Необязательно: Выберите команду Зарезервировать пространство хранения, чтобы оставить выделенное пространство памяти для таблицы.

Создание индекса

Создание индекса для одного или нескольких столбцов в таблице.

Чтобы создать индекс:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Раскройте в списке базу данных и выберите таблицу.
  3. В меню Действия выберите пункт Создать индекс.
  4. Следуйте инструкциям в мастере по созданию индексов.

  • Информация, связанная с данной
  • Индексы и вопросы их производительности
  • оператор CREATE INDEX

Включение и отключение индекса

Отключите индекс или включите отключенный индекс.

По умолчанию индекс включен. Сервер баз данных знает об индексе и обновляет его при операциях вставки, удаления и изменения. Если индекс отключить, он больше не будет обновляться при этих операциях.

Чтобы включить или отключить индекс:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Раскройте в списке базу данных, содержащую таблицу, к которой применяется индекс, и выберите эту таблицу.
  3. Раскройте панель Индексы.
  4. Выберите индекс.
  5. В меню Действия для панели выберите пункт Включить индекс или Отключить индекс.

  • Информация, связанная с данной
  • Как сервер баз данных обрабатывает отключенные индексы
  • Определения режимов объектов баз данных

Отбрасывание индекса

Отбросьте индекс, например, удалите индекс, который перестал использоваться эффективно.

Чтобы отбросить индекс:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Раскройте в списке базу данных, содержащую таблицу, к которой применяется индекс, и выберите эту таблицу.
  3. Раскройте панель Индексы.
  4. Выберите индекс.
  5. В меню Действия для панели выберите пункт Отбросить индекс.

  • Информация, связанная с данной
  • Отбрасывание индекса для таблицы - Эффективность изменения
  • оператор DROP INDEX

Загрузка и выгрузка данных с использованием внешних таблиц

Загрузка и выгрузка данных, которые хранятся и обрабатываются во внешних источниках, с использованием внешних таблиц.

Загрузка данных из внешней таблицы

Загрузите данные с помощью внешней таблицы в базу данных из источника, который является внешним для сервера базы данных.

Чтобы загрузить данные из внешней таблицы, выполните следующее:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Выберите таблицу базы данных, куда надо загрузить данные.

    Чтобы найти таблицу, начните вводить любую часть ее имени в поле Найти для вывода на экран соответствующих таблиц. Можно также раскрыть базу данных, чтобы вывести список ее таблиц.

  3. В меню Действия выберите пункт Загрузить таблицу.
  4. Следуйте инструкциям в мастере загрузки таблицы.

Выгрузка данных во внешнюю таблицу

Выгрузите данные из таблицы базы данных во внешнюю таблицу.

Чтобы выгрузить данные во внешнюю таблицу, выполните следующее:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Выберите таблицу базы данных, откуда надо выгрузить данные.

    Чтобы найти таблицу, начните вводить любую часть ее имени в поле Найти для вывода на экран соответствующих таблиц. Можно также раскрыть базу данных, чтобы вывести список ее таблиц.

  3. В меню Действия выберите пункт Выгрузить таблицу.
  4. Следуйте инструкциям в мастере выгрузки таблицы.

Создание внешней таблицы

Создание таблицы для загрузки и выгрузки данных, которые хранятся и обрабатываются вне сервера баз данных.

Требуемый компонент:
Для создания внешних таблиц требуется IBM Informix версии 11.50.xC6.

Чтобы создать внешнюю таблицу, выполните следующее:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Выберите базу данных, для которой нужно создать таблицу.
  3. В меню Действия выберите пункт Создать таблицу.
  4. Следуйте инструкциям в мастере создания таблицы.

    В поле Тип таблицы выберите тип внешней таблицы:

    • Внешняя фиксированная: Таблица в текстовом формате текста фиксированной ширины; управление данными такой таблицы и их хранение выполняется вне сервера баз данных Informix.
    • Внешняя с разделителями: Таблица в формате текста с разделителями; управление данными такой таблицы и их хранение выполняется вне сервера баз данных Informix.
    • Внешняя Informix: Таблица во внешнем формате Informix; управление данными такой таблицы и их хранение выполняется вне сервера баз данных Informix.

      Внешние таблицы используются для загрузки и выгрузки данных для баз данных.

Просмотр информации внешней таблицы

Просмотр информации о внешней таблице.

Чтобы просмотреть информацию о внешней таблице:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Чтобы найти таблицу, начните вводить любую часть ее имени в поле Найти для вывода на экран соответствующих таблиц. Можно также раскрыть базу данных, чтобы вывести список ее таблиц.
  3. Выберите таблицу в списке.

    На странице будет показана панель со сведениями о каждом объекте таблицы.

  4. Для вывода на экран дополнительных сведений об объекте таблицы щелкните кнопку раскрытия на панели объектов.

    На панелях Информация таблицы и Столбцы выводится информация о внешних таблицах.

Просмотр состояния задач загрузки и выгрузки

Просмотр состояния задач, загружающих и выгружающих данные с помощью внешних таблиц.

Чтобы просмотреть состояние задач загрузки и выгрузки:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Щелкните по базе данных или таблице, для которой запускается задача.
  3. Раскройте панель Информация.
  4. Нажмите вкладку Состояние задачи.
  5. В списке Показать выберите показываемый тип состояний: состояние всех задач, задач загрузки или задач выгрузки.

    Чтобы вывести дополнительную информацию о задаче, включая коды ошибок и возвращенные сообщения, нажмите для этой задачи кнопку подробностей.

  • Информация, связанная с данной
  • Сообщения об ошибках

Запуск сохраненных задач загрузки и выгрузки

Запуск задач, загружающих и выгружающих данные с помощью внешних таблиц.

Когда вы загружаете или выгружаете данные, используя внешние таблицы, соответствующая задача сохраняется. Эту задачу можно запустить еще раз или удалить.

Чтобы запустить сохраненную задачу загрузки или выгрузки:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Щелкните по базе данных или таблице, для которой запускается задача.
  3. Раскройте панель Информация.
  4. Нажмите вкладку Состояние задачи.
  5. В списке Показать выберите показываемый тип задач: Все задачи, Задачи загрузки или Задачи выгрузки.
  6. Выберите задачи для запуска.
  7. В меню Действия для панели выберите пункт Передать на выполнение.

Удаление задач загрузки и выгрузки

Удалите задачу загрузки или выгрузки или состояние задачи загрузки или выгрузки.

Когда вы загружаете или выгружаете данные, используя внешние таблицы, соответствующая задача сохраняется. Эту задачу можно запустить еще раз или удалить.

Удаляя задачу загрузки или выгрузки, вы удаляете также все описания состояния этой задачи. Но если вы удаляете состояние, сама задача не удаляется.

Чтобы удалить задачи загрузки или выгрузки:

  1. В меню для IBM OpenAdmin Tool (OAT) for Informix выберите SQL Toolbox → Менеджер схем.
  2. Щелкните по базе данных или таблице, для которой запускается задача.
  3. Раскройте панель Информация.
  4. Нажмите вкладку Состояние задачи.
  5. В списке Показать выберите тип удаляемой задачи.
    • Чтобы удалить только состояние, выберите Состояние всех задач, Состояние задач загрузки или Состояние задач выгрузки.
    • Чтобы удалить задачу и состояние, выберите Все задачи, Задачи загрузки или Задачи выгрузки.
  6. Выберите задачи для удаления.
  7. В меню Действия для панели выберите пункт Удалить.

Создание группы сохраненных задач загрузки и выгрузки

Сгруппируйте сохраненные задачи, чтобы их можно было запускать совместно.

Когда вы загружаете или выгружаете данные, используя внешние таблицы, соответствующая задача сохраняется. Вы можете запустить задачу еще раз, удалить ее или сгруппировать с другими сохраненными задачами.

Чтобы создать группу сохраненных задач загрузки и выгрузки:

  1. В меню OAT выберите SQL Toolbox → Менеджер схем.
  2. Щелкните по базе данных или таблице, для которой запускается задача.
  3. Раскройте панель Информация.
  4. Нажмите вкладку Состояние задачи.
  5. В списке Показать выберите показываемый тип задач: Все задачи, Задачи загрузки или Задачи выгрузки.
  6. Выберите задачи для включения в группу. Можно включить в одну группу и задачи загрузки, и задачи выгрузки.
  7. В меню Действия для панели щелкните по Создать группу и введите имя для группы.

Группы сохраненных задач загрузки и выгрузки для сервера выводятся на странице Состояние задач для базы данных sysadmin. Чтобы вывести группы на экран, выберите в списке баз данных базу данных sysadmin, раскройте панель Информация и щелкните по вкладке Состояние задачи.
