Informix Schema Manager Plug-in for OAT

O IBM® Informix® Schema Manager Plug-in for OpenAdmin Tool (OAT) fornece uma interface gráfica para visualizar e administrar bancos de dados e tabelas.

Com o Schema Manager plug-in, é possível executar estas tarefas:

  • Criar e eliminar bancos de dados, tabelas e índices.
  • Carregar e descarregar dados que são gerenciados e armazenados fora do servidor de banco de dados usando tabelas externas.
  • Monitorar informações sobre as estatísticas de tabelas e fragmentos.
  • Especificar se as estatísticas de distribuição de dados são calculadas apenas nos fragmentos que foram alterados em uma tabela fragmentada ou na tabela inteira.
  • Visualizar informações sobre objetos de banco de dados, incluindo procedimentos armazenados e funções, sequências, tipos de dados definidos pelo usuário (UDTs), privilégios, módulos DataBlade®, agregados, casts e classes operadoras.
  • Visualizar informações sobre objetos de tabela, incluindo colunas, índices, referências, privilégios, UDTs, restrições, fragmentos e acionadores.

Visualizando Objetos de Banco de Dados

Visualize informações sobre procedimentos armazenados e funções, sequências, tipos de dados definidos pelo usuário (UDTs), privilégios, módulos DataBlade, agregados, casts e classes operadoras.

Para visualizar objetos de banco de dados:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione o banco de dados na lista.

    A página exibe um painel de informações para cada um dos objetos de banco de dados.

  3. Para exibir mais detalhes sobre um objeto de banco de dados, clique no botão de expansão no painel do objeto.

Visualizando Objetos da Tabela

Visualize informações sobre colunas, índices, referências, privilégios, User-Defined Data Types (UDTs), limitadores, fragmentos e acionadores.

Para visualizar objetos da tabela:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Para localizar uma tabela, digite qualquer parte do nome da tabela no campo Localizar e clique em Procurar. Também é possível expandir um banco de dados para exibir as tabelas.
  3. Opcional: Selecione os filtros para incluir mais tabelas na lista:
    • Para incluir catálogos do sistema, selecione Incluir Catálogos.
    • Para incluir visualizações, selecione Incluir Visualizações. Uma visualização é uma tabela virtual baseada em uma instrução SELECT especificada.
    • Para incluir sinônimos, selecione Incluir Sinônimos. Um sinônimo é um nome alternativo para uma tabela, uma visualização, ou um objeto de sequência. Sinônimos se referem, normalmente, a tabelas que não estão no banco de dados atual.
  4. Selecione a tabela na lista.

    A página exibe um painel de informações para cada um dos objetos da tabela.

  5. Para exibir mais detalhes sobre um objeto da tabela, clique no botão de expansão no painel do objeto.

Revisando as Estatísticas de Tabela

Revise informações sobre as estatísticas de tabela e de fragmento, incluindo quando as estatísticas foram registradas pela última vez, a distribuição de dados e o percentual de linhas alteradas. Especifique as opções para atualizar automaticamente as estatísticas da tabela.

Pré-requisito:
O IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.70 para especificar as opções para atualizar automaticamente as estatísticas da tabela e exibir algumas das informações, por exemplo, o percentual de linhas alteradas.

Para revisar as estatísticas de tabela:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Expanda o banco de dados na lista.
  3. Selecione a tabela na lista.
  4. Expanda o painel Estatísticas.
  5. Para exibir as estatísticas do nível de fragmento, clique em Estatísticas de Fragmento. Essas informações são exibidas para o IBM Informix versão 11.70.

Criando um banco de dados

Crie um banco de dados e especifique o código de idioma do banco de dados.

Pré-requisito:
Para criar um banco de dados, o OpenAdmin Tool (OAT) for Informix requer o IBM Informix 11.70.xC2.


Para criar um banco de dados:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. No menu Ações, clique em Criar Banco de Dados.
  3. Siga as instruções no Assistente de Banco de Dados.

  • Conceitos Relacionados:
  • Use CREATE DATABASE (Guia de Design do Banco de Dados)
  • O Código de Idioma do Banco de Dados (Guia do Usuário do GLS)
  • Referência Relacionada:
  • Instrução CREATE DATABASE (Sintaxe de SQL)
  • criar argumento de banco de dados: Criar um banco de dados (API de administração de SQL) (Referência do Administrador)

Eliminando um Banco de Dados

Remova um banco de dados e exclua as tabelas e os dados armazenados nele.

Pré-requisito:
Para eliminar um banco de dados, o OpenAdmin Tool (OAT) for Informix requer o IBM Informix 11.70.xC2.


Para eliminar um banco de dados:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione o banco de dados.
  3. No menu Ações, clique em Eliminar Banco de Dados.

  • Referência Relacionada:
  • Instrução DROP DATABASE (Sintaxe de SQL)

Gerenciando Tabelas e Índices

Crie, descarte e trunque tabelas. Crie, desative e descarte índices.

Criando uma Tabela

Crie uma tabela e defina os atributos necessários para torná-la totalmente funcional.

Para criar uma tabela:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione o banco de dados no qual criar a tabela.
  3. No menu Ações, clique em Criar Tabela.
  4. Siga as direções no Assistente para Criar Tabela.

Descartando uma Tabela

Remova uma tabela com seus índices e dados associados.

Ao remover uma tabela, você também exclui os dados armazenados nela e quaisquer índices, restrições, acionadores, privilégios de acesso e sinônimos locais. Você também remove todas as visualizações baseadas na tabela e quaisquer tabelas de diagnóstico e violações associadas a ela.

Para descartar uma tabela:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Na lista, expanda o banco de dados que contém a tabela.
  3. Selecione a tabela.
  4. No menu Ações, clique em Descartar Tabela.

Truncando uma Tabela

Exclua todas as linhas de uma tabela e libere o espaço de armazenamento associado, ou reserve o espaço de armazenamento para reutilização pela tabela e seus índices.

A reutilização do espaço de armazenamento pode tornar o gerenciamento de armazenamento mais eficiente em aplicativos nos quais uma tabela é periodicamente esvaziada e recarregada com novas linhas.

Para truncar uma tabela:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Na lista, expanda o banco de dados que contém a tabela.
  3. Selecione a tabela.
  4. No menu Ações, clique em Truncar Tabela.
  5. Opcional: Selecione Reservar o Espaço de Armazenamento para manter o espaço de armazenamento alocado para a tabela.

Criando um Índice

Crie um índice para uma ou mais colunas em uma tabela.

Para criar um índice:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Na lista, expanda o banco de dados e selecione a tabela.
  3. No menu Ações, clique em Criar Índice.
  4. Siga as orientações do assistente Criar Índice.

  • Conceitos Relacionados:
  • Considerações sobre Índices e Desempenho do Índice (Guia de Desempenho)
  • Referência Relacionada:
  • Instrução CREATE INDEX (Sintaxe de SQL)

Desativando ou Ativando um Índice

Desative um índice ou ative um índice que estava desativado.

Um índice é ativado por padrão. O servidor de banco de dados reconhece o índice e o atualiza durante as operações de inserção, exclusão e atualização. Quando um índice é desativado, ele não é mais atualizado durante essas operações.

Para ativar ou desativar um índice:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Na lista, expanda o banco de dados que contém a tabela à qual o índice se aplica e selecione a tabela.
  3. Expanda o painel Índices.
  4. Selecione o índice.
  5. No menu Ações do painel, clique em Ativar Índice ou Desativar Índice.

  • Conceitos Relacionados:
  • Como o Servidor de Banco de Dados Trata Índices Desativados (Sintaxe de SQL)
  • Definições de Modos de Objeto de Banco de Dados (Sintaxe de SQL)

Descartando um Índice

Descarte um índice; por exemplo, para remover um índice que não seja mais eficiente.

Para descartar um índice:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Na lista, expanda o banco de dados que contém a tabela à qual o índice se aplica e selecione a tabela.
  3. Expanda o painel Índices.
  4. Selecione o índice.
  5. No menu Ações do painel, clique em Descartar Índice.

  • Tarefas Relacionadas:
  • Eliminando Índices para Eficiência de Atualização de Tabela (Guia de Desempenho)
  • Referência Relacionada:
  • Instrução DROP INDEX (Sintaxe de SQL)

Carregando e Descarregando Dados com Tabelas Externas

Carregar e descarregar dados que são gerenciados e armazenados fora do servidor de banco de dados usando tabelas externas.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Carregando Dados com uma Tabela Externa

Carregue dados na tabela de banco de dados a partir de uma origem externa para o servidor de banco de dados usando uma tabela externa.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Para carregar dados com uma tabela externa:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione a tabela de banco de dados na qual os dados serão carregados.

    Para localizar uma tabela, digite qualquer parte do nome da tabela no campo Localizar e clique em Procurar. Também é possível expandir um banco de dados para exibir as tabelas.

  3. No menu Ações, clique em Carregar Tabela.
  4. Siga as orientações no assistente para Carregar Tabela.

Descarregando Dados para uma Tabela Externa

Descarregue dados de uma tabela de banco de dados para uma tabela externa.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Para descarregar dados para uma tabela externa:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione a tabela de banco de dados da qual os dados serão descarregados.

    Para localizar uma tabela, digite qualquer parte do nome da tabela no campo Localizar e clique em Procurar. Também é possível expandir um banco de dados para exibir as tabelas.

  3. No menu Ações, clique em Descarregar Tabela.
  4. Siga as orientações no assistente para Descarregar Tabela.

Criando uma Tabela Externa

Crie uma tabela que seja gerenciada e armazenada fora do servidor de banco de dados para carregar e descarregar dados.

Pré-requisito:
Para criar tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6.

Para criar uma tabela externa:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Selecione o banco de dados no qual criar a tabela.
  3. No menu Ações, clique em Criar Tabela.
  4. Siga as direções no Assistente para Criar Tabela.

    Para Tipo de Tabela, selecione um tipo de tabela externa:

    • Fixa Externa: Uma tabela em formato de texto de largura fixa cujos dados são gerenciados e armazenados fora do servidor de banco de dados Informix.
    • Delimitada Externa: Uma tabela em formato de texto delimitado cujos dados são gerenciados e armazenados fora do servidor de banco de dados Informix.
    • Informix Externa: Uma tabela em formato Informix interno cujos dados são gerenciados e armazenados fora do servidor de banco de dados Informix.

      As tabelas externas são usadas para carregar e descarregar dados do seu banco de dados.

Visualize informações sobre uma tabela externa.

Essas informações são exibidas para o Informix 11.50.xC6 ou mais recente.

Para visualizar informações sobre uma tabela externa:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Para localizar uma tabela, digite qualquer parte do nome da tabela no campo Localizar e clique em Procurar. Também é possível expandir um banco de dados para exibir as tabelas.
  3. Selecione a tabela na lista.

    A página exibe um painel de informações para cada um dos objetos da tabela.

  4. Para exibir mais detalhes sobre um objeto da tabela, clique no botão de expansão no painel do objeto.

    Os painéis Informações sobre a Tabela e Colunas exibem informações específicas para tabelas externas.

Visualizando o Status de Tarefas de Carregamento e de Descarregamento

Visualize os status das tarefas enviadas por meio do IBM OpenAdmin Tool (OAT) for Informix para carregar e descarregar dados usando tabelas externas.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Para visualizar o status de tarefas de carregamento e descarregamento:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Clique no banco de dados ou na tabela em que a tarefa é executada.
  3. Expanda o painel Informações.
  4. Clique na guia Status da Tarefa.

    Para exibir informações adicionais sobre uma tarefa, incluindo os códigos de erro e as mensagens de retorno, clique no ícone de detalhes da tarefa.

  • Conceitos Relacionados:
  • Mensagens de Erro (Referência do Administrador)

Executando Tarefas de Carregamento e de Descarregamento Salvas

Execute tarefas que carregam e descarregam dados usando tabelas externas.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Quando se carrega ou descarrega dados usando tabelas externas, a tarefa é salva. É possível executar a tarefa novamente ou excluí-la.

Para executar uma tarefa de carregamento e de descarregamento salva:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Clique no banco de dados ou na tabela em que a tarefa é executada.
  3. Expanda o painel Informações.
  4. Clique na guia Tarefas.
  5. Selecione as tarefas a serem executadas.
  6. No menu Ações do painel, clique em Submeter.

Excluindo Tarefas de Carregamento e de Descarregamento

Exclua uma tarefa de carregamento e de descarregamento ou o status de uma tarefa de carregamento ou de descarregamento.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Quando você carrega ou descarrega dados usando a OAT, a tarefa é salva. É possível enviar a tarefa novamente ou excluí-la. É possível também excluir o status das tarefas que são enviadas.

Importante:
Quando você exclui uma tarefa na página Tarefa, os resultados da tarefa também são excluídos da página Status de Tarefa. Quando você exclui um status de tarefa, somente o status de tarefa é excluído.

Para excluir tarefas de carregamento e de descarregamento:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Clique no banco de dados ou na tabela em que a tarefa é executada.
  3. Expanda o painel Informações.
  4. Selecione a guia apropriada:
    • Para excluir a tarefa e o status de tarefa, clique na guia Tarefas.
    • Para excluir apenas o status de tarefa, clique na guia Status de Tarefa.
  5. Selecione a tarefa.
  6. Clique em Ações > Excluir.

Criando um Grupo de Tarefas de Carregamento ou de Descarregamento Salvas

Agrupe tarefas salvas para que possam ser submetidas juntas.

Pré-requisito:
Para carregar e descarregar dados com tabelas externas, o IBM OpenAdmin Tool (OAT) for Informix requer o IBM Informix versão 11.50.xC6 ou mais recente.

Quando você carrega ou descarrega dados usando a OAT, a tarefa é salva. É possível executar a tarefa novamente, excluí-la ou agrupá-la com outras tarefas salvas.

Para criar um grupo de tarefas de carregamento ou de descarregamento salvas:

  1. No menu OAT, clique em Caixa de Ferramentas SQL > Schema Manager.
  2. Clique no banco de dados ou na tabela em que a tarefa é executada.
  3. Expanda o painel Informações.
  4. Clique na guia Tarefas.
  5. Selecione as tarefas a serem incluídas no grupo. As tarefas de carregamento e de descarregamento podem ser incluídas no mesmo grupo.
  6. No menu Ações do painel, clique em Criar Grupo e insira um nome para o grupo.
