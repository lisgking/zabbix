Plug-in de Informix Schema Manager para OAT

El Plug-in de IBM® Informix® Schema Manager proporciona una interfaz gráfica para visualizar y administrar bases de datos y tablas.

Con el plug-in Gestor de esquemas, puede realizar las siguientes tareas:

  • Crear y descartar bases de datos, tablas e índices.
  • Cargar y descargar datos gestionados y almacenados fuera del servidor de base de datos utilizando tablas externas.
  • Supervisar información sobre las estadísticas de tablas y fragmentos.
  • Especificar si las estadísticas de distribución de datos se calculan sólo en los fragmentos que cambian en una tabla fragmentada o en toda la tabla.
  • Ver información sobre objetos de bases de datos incluyendo procedimientos y funciones almacenadas, secuencias, tipos de datos definidos por el usuario (UDT), privilegios, módulos de DataBlade®, agregados, conversiones y clases de operador.
  • Ver información sobre objetos de tablas incluyendo columnas, índices, referencias, privilegios, UDT, restricciones, fragmentos y desencadenantes.

Visualización de objetos de bases de datos

Ver información sobre procedimientos y funciones almacenadas, secuencias, tipos de datos definidos por el usuario (UDT), privilegios, módulos de DataBlade, agregados, conversiones y clases de operador.

Para ver objetos de bases de datos:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. Seleccione la base de datos en la lista.

    La página muestra un panel de información para cada objeto de base de datos.

  3. Para ver más detalles sobre un objeto de base de datos, pulse el botón de ampliación en el panel del objeto.

Visualización de objetos de tablas

Ver información sobre columnas, índices, referencias, privilegios, tipos de datos definidos por el usuario (UDT), restricciones, fragmentos y desencadenantes.

Para ver objetos de tablas:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. Para encontrar una tabla, especifique cualquier parte del nombre de la tabla en el campo Buscar y pulse Buscar. También puede ampliar una base de datos para ver las tablas.
  3. Opcional: Seleccione los filtros para incluir más tablas en la lista:
    • Para incluir catálogos de sistema, seleccione Incluir catálogos.
    • Para incluir vistas, seleccione Incluir vistas. Una vista es una tabla virtual basada en una sentencia SELECT especificada.
    • Para incluir sinónimos, seleccione Incluir sinónimos. Un sinónimo es un nombre alternativo para un objeto de tabla, de vista o de secuencia. Normalmente, los sinónimos hacen referencia a tablas que no están en la base de datos actual.
  4. Seleccione la tabla en la lista.

    La página muestra un panel de información para cada objeto de tabla.

  5. Para ver más detalles sobre un objeto de tabla, pulse el botón de ampliación en el panel del objeto.

Revisión de estadísticas de tabla

Revise la información sobre las estadísticas de tabla y fragmento, incluyendo cuándo se registraron por última vez las estadísticas, la distribución de datos y el porcentaje de filas cambiadas. Especifique las opciones para actualizar automáticamente las estadísticas de la tabla.

Requisito previo:
IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.70 para especificar las opciones para actualizar automáticamente las estadísticas de la tabla y para visualizar cierta información, por ejemplo, el porcentaje de filas cambiadas.

Para revisar las estadísticas de tabla:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Amplíe la base de datos en la lista.
  3. Seleccione la tabla en la lista.
  4. Amplíe el panel Estadísticas.
  5. Para mostrar las estadísticas del nivel de los fragmentos, pulse Estadísticas de fragmento. Esta información se muestra para IBM Informix versión 11.70.

Creación de una base de datos

Cree una base de datos y especifique el entorno local de base de datos.

Requisito previo:
Para crear una base de datos, OpenAdmin Tool (OAT) para Informix requiere IBM Informix 11.70.xC2.


Para crear una base de datos:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En el menú Acciones, pulse Crear base de datos.
  3. Siga las indicaciones del asistente Crear base de datos.

  • Conceptos relacionados:
  • Utilizar CREATE DATABASE (Guía de diseño de bases de datos)
  • El entorno local de la base de datos (guía del usuario de GLS)
  • Referencia relacionada:
  • Sentencia CREATE DATABASE (sintaxis SQL)
  • Crear argumento de base de datos: Crear una base de datos (API de administración SQL) (referencia del administrador)

Descartar una base de datos

Elimine una base de datos y suprima las tablas y los datos almacenados en ella.

Requisito previo:
Para descartar una base de datos, OpenAdmin Tool (OAT) para Informix requiere IBM Informix 11.70.xC2.


Para descartar una base de datos:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. Seleccione la base de datos.
  3. En el menú Acciones, pulse Descartar base de datos.

  • Referencia relacionada:
  • La sentencia DROP DATABASE (Sintaxis SQL)

Gestión de tablas e índices

Crear, eliminar y truncar tablas. Crear, inhabilitar y eliminar índices.

Creación de una tabla

Crear una tabla y definir los atributos necesarios para hacer que la tabla sea totalmente funcional.

Para crear una tabla:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. Seleccione la base de datos en la que crear la tabla.
  3. En el menú Acciones, pulse Crear tabla.
  4. Siga las pautas del asistente Crear tabla.

Eliminación de una tabla

Eliminar una tabla con sus índices y datos correspondientes.

Cuando suprime una tabla, también elimina los datos almacenados en ella y cualquier índice, restricción, desencadenante, privilegio de acceso y sinónimo local. También suprime todas las vistas basadas en la tabla y cualquier tabla de violaciones y diagnósticos.

Para eliminar una tabla:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En la lista, amplíe la base de datos que contiene la tabla.
  3. Seleccione la tabla.
  4. En el menú Acciones, pulse Eliminar tabla.

Truncamiento de una tabla

Suprimir todas las filas de una tabla y liberar el espacio de almacenamiento correspondiente o reservar el espacio de almacenamiento para su reutilización por la tabla y sus índices.

La reutilización del espacio de almacenamiento puede hacer que la administración de almacenamiento sea más eficaz en aplicaciones en las que una tabla se vacía y se recarga con filas nuevas periódicamente.

Para truncar una tabla:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En la lista, amplíe la base de datos que contiene la tabla.
  3. Seleccione la tabla.
  4. En el menú Acciones, pulse Truncar tabla.
  5. Opcional: Seleccione Reservar el espacio de almacenamiento para mantener el espacio de almacenamiento asignado a la tabla.

Creación de un índice

Crear un índice para una o más columnas de una tabla.

Para crear un índice:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En la lista, amplíe la base de datos y seleccione la tabla.
  3. En el menú Acciones, pulse Crear índice.
  4. Siga las pautas del asistente Crear índice.

  • Conceptos relacionados:
  • Índices y consideraciones sobre el rendimiento de índices (guía de rendimiento)
  • Referencia relacionada:
  • La sentencia CREATE INDEX (Sintaxis SQL)

Inhabilitación o habilitación de un índice

Inhabilitar un índice o habilitar un índice que estaba inhabilitado.

De forma predeterminada, un índice está habilitado. El servidor de base de datos reconoce el índice y lo actualiza durante las operaciones de inserción, eliminación y actualización. Cuando se inhabilita un índice, ya no se actualiza durante estas operaciones.

Para habilitar o inhabilitar un índice:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En la lista, amplíe la base de datos que contenga la tabla a la que se aplica el índice para seleccionar la tabla.
  3. Amplíe el panel Índices.
  4. Seleccione el índice.
  5. En el menú Acciones para el panel, pulse Habilitar índice o Inhabilitar índice.

  • Conceptos relacionados:
  • Cómo trata el servidor de base de datos los índices inhabilitados(Sintaxis SQL)
  • Definiciones de las modalidades de objetos de la base de datos (Sintaxis SQL)

Eliminación de un índice

Eliminar un índice; por ejemplo, eliminar un índice que ya no es eficaz.

Para eliminar un índice:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. En la lista, amplíe la base de datos que contenga la tabla a la que se aplica el índice para seleccionar la tabla.
  3. Amplíe el panel Índices.
  4. Seleccione el índice.
  5. En el menú Acciones para el panel, pulse Eliminar índice.

  • Tareas relacionadas:
  • Descartar índices para mejorar la eficiencia de la actualización de tabla (Guía de rendimiento)
  • Referencia relacionada:
  • La sentencia DROP INDEX (Sintaxis SQL)

Carga y descarga de datos con tablas externas

Cargar y descargar datos gestionados y almacenados fuera del servidor de base de datos utilizando tablas externas.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Carga de datos con una tabla externa

Cargar datos en una tabla de base de datos desde una fuente externa al servidor de la base de datos utilizando una tabla externa.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Para cargar datos con una tabla externa:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Seleccione la tabla de base de datos a la que cargar datos.

    Para encontrar una tabla, especifique cualquier parte del nombre de la tabla en el campo Busar y pulse Buscar. También puede ampliar una base de datos para ver las tablas.

  3. En el menú Acciones, pulse Cargar tabla.
  4. Siga las pautas del asistente Cargar tabla.

Descarga de datos a una tabla externa

Descargar datos de una tabla de base de datos a una tabla externa.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Para descargar datos a una tabla externa:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Seleccione la tabla de base de datos de la que descargar datos.

    Para encontrar una tabla, especifique cualquier parte del nombre de la tabla en el campo Busar y pulse Buscar. También puede ampliar una base de datos para ver las tablas.

  3. En el menú Acciones, pulse Descargar tabla.
  4. Siga las pautas del asistente Descargar tabla.

Creación de una tabla externa

Cree una tabla que se gestione y almacene fuera del servidor de datos para cargar y descargar datos.

Requisito previo:
Para crear tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6.

Para crear una tabla externa:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Seleccione la base de datos para la que crear la tabla.
  3. En el menú Acciones, pulse Crear tabla.
  4. Siga las pautas del asistente Crear tabla.

    Para Tipo de tabla, seleccione un tipo de tabla externo:

    • Externa fija: Tabla en formato de texto de anchura fija cuyos datos se gestionan y almacenan fuera del servidor de base de datos de Informix.
    • Externa delimitada: Tabla en formato de texto delimitado cuyos datos se gestionan y almacenan fuera del servidor de base de datos de Informix.
    • Informix externa: Tabla en formato de texto interno de Informix cuyos datos se gestionan y almacenan fuera del servidor de base de datos de Informix.

      Las tablas externas se utilizan para cargar y descargar datos para la base de datos.

Visualización de la información de tablas externas

Ver información sobre una tabla externa.

Esta información se visualiza para Informix 11.50.xC6 o posterior.

Para ver información sobre una tabla externa:

  1. En el menú de IBM OpenAdmin Tool (OAT) para Informix, pulse SQL Toolbox > Schema Manager.
  2. Para encontrar una tabla, especifique cualquier parte del nombre de la tabla en el campo Busar y pulse Buscar. También puede ampliar una base de datos para ver las tablas.
  3. Seleccione la tabla en la lista.

    La página muestra un panel de información para cada objeto de tabla.

  4. Para ver más detalles sobre un objeto de tabla, pulse el botón de ampliación en el panel del objeto.

    El panel Información de la tabla y el panel Columnas muestran información específica de las tablas externas.

Visualización del estado de las tareas de carga y descarga

Ver el estado de las tareas sometidas mediante IBM OpenAdmin Tool (OAT) para Informix para cargar y descargar datos utilizando tablas externas.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Para ver el estado de las tareas de carga y descarga:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Pulse la base de datos o la tabla en la que se ejecuta la tarea.
  3. Amplíe el panel Información.
  4. Pulse la ficha Estado de la tarea.

    Para mostrar más información sobre una tarea, incluidos los códigos de error y los mensajes de retorno, pulse el icono de detalles de la tarea.

  • Conceptos relacionados:
  • Mensajes de error (Guía del administrador)

Ejecución de tareas de carga y descarga guardadas

Ejecutar tareas que cargan y descargan datos utilizando tablas externas.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Cuando se carga o descargan datos utilizando tablas externas, la tarea se guarda. Puede volver a ejecutar la tarea o eliminarla.

Para ejecutar una tarea de carga o descarga guardada:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Pulse la base de datos o la tabla en la que se ejecuta la tarea.
  3. Amplíe el panel Información.
  4. Pulse el separador Tareas.
  5. Seleccione las tareas que desee ejecutar.
  6. En el menú Acciones para el panel, pulse Enviar.

Supresión de tareas de carga y descarga

Suprimir una tarea de carga o descarga, o el estado de una tarea de carga o descarga.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Al cargar o descargar datos utilizando OAT, la tarea se guarda. Puede volver a enviar la tarea o eliminarla. También puede suprimir el estado de las tareas enviadas.

Importante:
Cuando se suprime una tarea en la página Tarea, los resultados de la tarea también se suprimen de la página Estado de tarea. Cuando se suprime un estado de tarea, sólo se suprime el estado de tarea.

Para suprimir tareas de carga y descarga:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Pulse la base de datos o la tabla en la que se ejecuta la tarea.
  3. Amplíe el panel Información.
  4. Seleccione el separador correspondiente:
    • Para suprimir la tarea y el estado de la tarea, pulse el separador Tareas.
    • Para suprimir sólo el estado de la tarea, pulse el separador Estado de tarea.
  5. Seleccione la tarea.
  6. Pulse Acciones > Suprimir.

Creación de un grupo de tareas guardadas de carga y descarga

Agrupar tareas guardadas de manera que se puedan enviar juntas.

Requisito previo:
Para cargar y descargar datos con tablas externas, IBM OpenAdmin Tool (OAT) para Informix requiere IBM Informix versión 11.50.xC6 o posterior.

Al cargar o descargar datos utilizando OAT, la tarea se guarda. Puede ejecutar de nuevo la tarea, eliminarla o agruparla con otras tareas guardadas.

Para crear un grupo de tareas de carga o descarga guardadas:

  1. En el menú OAT, pulse SQL Toolbox >Gestor de esquemas.
  2. Pulse la base de datos o la tabla en la que se ejecuta la tarea.
  3. Amplíe el panel Información.
  4. Pulse el separador Tareas.
  5. Seleccione las tareas que desee incluir en el grupo. Las tareas de carga y descarga se pueden incluir en el mismo grupo.
  6. En el menú Acciones para el panel, pulse Crear grupo y escriba un nombre para el grupo.
