OAT 用の Informix スキーマ マネージャ プラグイン

OpenAdmin Tool (OAT) 用の IBM® Informix® スキーマ マネージャ プラグインは、データベースと表を表示および管理するためのグラフィカル インターフェイスを提供します。

スキーマ マネージャ プラグインを使用すると、次のタスクを実行できます。

  • データベース、表、およびインデックスの作成と削除。
  • データベース サーバの外部で管理および保管されているデータを、外部表を使用してロードおよびアンロードします。
  • 表とフラグメントに関する統計情報の監視。
  • データ分散統計の計算対象を、フラグメント表内の変更されたフラグメントのみにするか、あるいは表全体にするかの指定。
  • ストアド プロシジャおよび関数、シーケンス、ユーザ定義データ型 (UDT)、アクセス権、DataBlade® モジュール、集計関数、キャスト、および演算子クラスなどのデータベース オブジェクトに関する情報の表示。
  • 列、インデックス、リファレンス、アクセス権、UDT、制約、フラグメント、トリガなどの表オブジェクトに関する情報の表示。

データベース オブジェクトの表示

ストアド プロシジャおよび関数、シーケンス、ユーザ定義データ型 (UDT)、アクセス権、DataBlade モジュール、集計関数、キャスト、および演算子クラスに関する情報を表示します。

データベース オブジェクトを表示するには

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リスト内のデータベースを選択します。

    ページには、各データベース オブジェクトの情報のパネルが表示されます。

  3. データベース オブジェクトの詳細を表示するには、オブジェクト パネルにある展開ボタンをクリックします。

表オブジェクトの表示

列、インデックス、リファレンス、アクセス権、ユーザ定義データ型 (UDT)、制約、フラグメント、トリガに関する情報を表示します。

表オブジェクトを表示するには

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. 表を検索するには、「検索」 フィールドに表名の一部を入力してから、「検索」をクリックします。 データベースを展開して表を表示させることもできます。
  3. オプション: 次のようにフィルタを選択して、リストに表を追加します。
    • システム カタログを含めるには、「カタログを含める」を選択します。
    • ビューを含めるには、「ビューを含める」を選択します。 ビューは、指定した SELECT 文に基づく仮想表です。
    • シノニムを含めるには、「シノニムを含める」を選択します。 シノニムは、表、ビュー、またはシーケンス オブジェクトの代替名です。一般的に、シノニムは現行のデータベースにはない表を参照します。
  4. リスト内の表を選択します。

    ページには、各表オブジェクトの情報のパネルが表示されます。

  5. 表オブジェクトの詳細を表示するには、オブジェクト パネルにある展開ボタンをクリックします。

表の統計情報の確認

統計情報が最後に記録された日時、データ分散、および変更された行のパーセンテージなどを含め、表とフラグメントの統計情報を確認します。 表の統計情報の自動更新オプションを指定します。

前提条件:
表の統計情報の自動更新オプションの指定、および変更された行のパーセンテージなどの一部の情報の表示を行う場合、IBM OpenAdmin Tool (OAT) for Informix には、IBM Informix バージョン 11.70 が必要です。

表の統計情報を確認するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リスト内のデータベースを展開します。
  3. リスト内の表を選択します。
  4. 「統計」 パネルを展開します。
  5. フラグメント レベルの統計情報を表示するには、「フラグメント統計」をクリックします。この情報は、IBM Informix バージョン 11.70 に対して表示されます。

データベースの作成

データベースを作成し、データベース ロケールを指定します。

前提条件:
データベースを作成する場合、 OpenAdmin Tool (OAT) for Informix には IBM Informix 11.70.xC2 が必要です。


データベースを作成するには、次の手順で行います。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. 「操作」メニューで、「データベースの作成」をクリックします。
  3. 「データベースの作成」ウィザードの指示に従います。

データベースの削除

データベースを削除し、表とそこに格納されたデータを削除します。

前提条件:
データベースを削除する場合、 OpenAdmin Tool (OAT) for Informix には IBM Informix 11.70.xC2 が必要です。


データベースを削除するには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. データベースを選択します。
  3. 「操作」メニューで、「データベースの削除」をクリックします。

  • 関連資料
  • DROP DATABASE 文 (SQL 構文)

表およびインデックスの管理

表を作成、削除、および切り捨てます。インデックスを作成、無効化、および削除します。

表の作成

表を作成して、表を完全に機能させるために必要な属性を定義します。

表を作成するには

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. 表を作成するデータベースを選択します。
  3. 「アクション」メニューで、「表の作成」をクリックします。
  4. 「表の作成」ウィザードの手順に従ってください。

表の削除

表を、その関連するインデックスおよびデータとともに削除します。

表を削除すると、表に格納されているデータ、インデックス、制約、トリガ、アクセス権、およびローカル シノニムも削除されます。 また、表に基づいているすべてのビュー、表に関連付けられている違反表および診断表も削除されます。

表を削除するには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リストで、表が含まれるデータベースを展開します。
  3. 表を選択します。
  4. 「操作」メニューで、「表の削除」をクリックします。

表の切捨て

表からすべての行を削除して、関連する格納域を解放するか、表およびそのインデックスで再利用できるように格納域を予約します。

表が定期的に空にされ、新規行とともに再ロードされるアプリケーションでは、格納域を再利用することで、格納域管理を効率的に行うことができます。

表を切り捨てるには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リストで、表が含まれるデータベースを展開します。
  3. 表を選択します。
  4. 「操作」メニューで、「表の切捨て」をクリックします。
  5. オプション：「格納領域を予約する」を選択して、表に割り当てられる格納域を確保します。

インデックスの作成

表の 1 つ以上の列に対してインデックスを作成します。

インデックスを作成するには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リストでデータベースを展開し、表を選択します。
  3. 「操作」メニューで、「インデックスの作成」をクリックします。
  4. 「インデックスの作成」ウィザードの指示に従います。

  • 関連概念:

  • 関連資料
  • CREATE INDEX 文 (SQL 構文)

インデックスの無効化または有効化

インデックスを無効にします。または無効化されているインデックスを有効にします。

インデックスは、デフォルトでは有効になっています。データベース サーバは、インデックスを認識し、挿入、削除、および更新の各操作時にインデックスを更新します。インデックスを無効にすると、これらの操作時に更新は実行されなくなります。

インデックスを有効または無効にするには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リストで、インデックスの適用先の表が含まれるデータベースを展開し、表を選択します。
  3. 「インデックス」パネルを展開します。
  4. インデックスを選択します。
  5. パネルの「操作」メニューで、「インデックスの有効化」または「インデックスの無効化」をクリックします。

  • 関連概念:
  • データベース サーバによる停止モードのインデックスの処理 (SQL 構文)
  • データベース オブジェクト モードの定義 (SQL 構文)

インデックスの削除

インデックスを削除します (例えば、効率が悪くなったインデックスを除去するため)。

インデックスを削除するには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. リストで、インデックスの適用先の表が含まれるデータベースを展開し、表を選択します。
  3. 「インデックス」パネルを展開します。
  4. インデックスを選択します。
  5. パネルの「操作」メニューで、「インデックスの削除」をクリックします。

外部表を使用したデータのロードおよびアンロード

データベース サーバの外部で管理および保管されているデータを、外部表を使用してロードおよびアンロードします。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以降が必要です。

外部表でのデータのロード

外部表を使用して、データベース サーバに対して外部であるソースからデータベース表にデータをロードします。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

外部表でデータをロードするには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. データをロードするデータベース表を選択します。

    表を検索するには、「検索」 フィールドに表名の一部を入力してから、「検索」をクリックします。 データベースを展開して表を表示させることもできます。

  3. 「操作」メニューで、「表のロード」をクリックします。
  4. 「表のロード」ウィザードの指示に従います。

外部表でのデータのアンロード

データベース表から外部表にデータをアンロードします。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

外部表にデータをアンロードするには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. データをアンロードするデータベース表を選択します。

    表を検索するには、「検索」 フィールドに表名の一部を入力してから、「検索」をクリックします。 データベースを展開して表を表示させることもできます。

  3. 「操作」メニューで、「表のアンロード」をクリックします。
  4. 「表のアンロード」ウィザードの指示に従います。

外部表の作成

データのロードおよびアンロードを行うため、データベース サーバの外部で管理および保管される表を作成します。

前提条件:
外部表を作成する場合、IBM OpenAdmin Tool (OAT) for Informixには IBM Informix バージョン 11.50.xC6 が必要です。

外部表を作成するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. 表を作成する対象のデータベースを選択します。
  3. 「アクション」メニューで、「表の作成」をクリックします。
  4. 「表の作成」ウィザードの手順に従ってください。

    「表タイプ」で、外部表のタイプを選択します。

    • 外部 (固定): Informix データベース サーバ外部でデータが管理および保管される固定長テキスト形式の表。
    • 外部 (区切り記号付き): Informix データベース サーバ外部でデータが管理および保管される区切りテキスト形式の表。
    • 外部 Informix: Informix データベース サーバ外部でデータが管理および保管される内部 Informix 形式の表。

      外部表は、データベースのデータのロードとアンロードに使用されます。

外部表情報の表示

外部表に関する情報を表示します。

この情報は、Informix 11.50.xC6 以降に対して表示されます。

外部表に関する情報を表示するには、次の手順を実行します。

  1. IBM OpenAdmin Tool (OAT) for Informix のメニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. 表を検索するには、「検索」 フィールドに表名の一部を入力してから、「検索」をクリックします。 データベースを展開して表を表示させることもできます。
  3. リスト内の表を選択します。

    ページには、各表オブジェクトの情報のパネルが表示されます。

  4. 表オブジェクトの詳細を表示するには、オブジェクト パネルにある展開ボタンをクリックします。

    「表情報」パネルおよび「列」パネルに、外部表固有の情報が表示されます。

ロード タスクおよびアンロード タスクの状態の表示

外部表を使用してデータをロードおよびアンロードするために IBM OpenAdmin Tool (OAT) for Informix を介して実行要求されるタスクの状態を表示します。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

ロード タスクおよびアンロード タスクの状態を表示するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. タスクの実行対象のデータベースまたは表をクリックします。
  3. 「情報」パネルを展開します。
  4. 「タスクの状態」タブをクリックします。

    エラー コードや戻りメッセージなど、タスクに関する詳細情報を表示するには、タスクの詳細アイコンをクリックします。

  • 関連概念:
  • エラー メッセージ (管理者リファレンス)

保存されているロード タスクおよびアンロード タスクの実行

外部表を使用して、データをロードするタスクおよびアンロードするタスクを実行します。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

外部表を使用してデータをロードまたはアンロードすると、タスクが保存されます。そのタスクを再び実行すること、または削除することができます。

保存されているロード タスクまたはアンロード タスクを実行するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. タスクの実行対象のデータベースまたは表をクリックします。
  3. 「情報」パネルを展開します。
  4. 「タスク」タブをクリックします。
  5. 実行するタスクを選択します。
  6. パネルの「操作」メニューで、「実行要求」をクリックします。

ロード タスクおよびアンロード タスクの削除

ロード タスクまたはアンロード タスク、あるいはロード タスクの状態またはアンロード タスクの状態を削除します。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

OAT を使用してデータをロードまたはアンロードすると、タスクが保存されます。 そのタスクを再び実行要求すること、または削除することができます。また、実行要求したタスクの状態を削除することもできます。

重要:
「タスク」ページでタスクを削除すると、タスクの結果は「タスク状況」ページからも削除されます。 タスク状況を削除すると、タスク状況のみが削除されます。

ロード タスクおよびアンロード タスクを削除するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. タスクの実行対象のデータベースまたは表をクリックします。
  3. 「情報」パネルを展開します。
  4. 次のいずれか該当するタブを選択します。
    • タスクとタスク状況を削除するには、「タスク」タブをクリックします。
    • タスク状況のみを削除するには、「タスク状況」タブをクリックします。
  5. タスクを選択します。
  6. 「操作」 > 「削除」をクリックします。

保存済みロード タスクまたはアンロード タスクのグループの作成

保存されているタスクを同時に実行できるようにするために、それらのタスクをグループ化します。

前提条件:
外部表を使用してデータをロードおよびアンロードするため、IBM OpenAdmin Tool (OAT) for Informix には IBM Informix バージョン 11.50.xC6 以上が必要です。

OAT を使用してデータをロードまたはアンロードすると、タスクが保存されます。 そのタスクを再び実行すること、削除すること、または他の保存済みタスクとともにグループ化することができます。

保存済みロード タスクまたはアンロード タスクのグループを作成するには、次の手順を実行します。

  1. OAT メニューで、「SQL Toolbox」 > 「スキーマ マネージャ」をクリックします。
  2. タスクの実行対象のデータベースまたは表をクリックします。
