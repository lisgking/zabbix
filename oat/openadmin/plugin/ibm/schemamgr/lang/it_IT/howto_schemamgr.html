Informix Schema Manager Plug-in for OAT

IBM® Informix® Schema Manager Plug-in for OpenAdmin Tool (OAT) fornisce un'interfaccia grafica per la visualizzazione e l'amministrazione di database e tabelle.

Con il plug-in Schema Manager è possibile eseguire queste attività:

  • Creare e rimuovere database, tabelle e indici.
  • Caricare e scaricare i dati gestiti e memorizzati al di fuori del server di database utilizzando le tabelle esterne.
  • Monitorare le informazioni sulle statistiche per le tabelle e i frammenti.
  • Specificare se le statistiche di distribuzione dei dati debbano essere calcolate solo in base ai frammenti modificati in una tabella frammentata o nell'intera tabella.
  • Visualizzare le informazioni sugli oggetti del database, incluse le procedure memorizzate e le funzioni, le sequenze, i tipi di dati definiti dall'utente (UDT), i privilegi, i moduli DataBlade®, gli aggregati, i cast e le classi operatore.
  • Visualizzare le informazioni sugli oggetti della tabella, incluse le colonne, gli indici, i riferimenti, i privilegi, gli UDT, i vincoli, i frammenti e i trigger.

Visualizzazione degli oggetti del database

Visualizzare le informazioni relative a procedure memorizzate e funzioni, sequenze, tipi di dati definiti dall'utente (UDT), privilegi, moduli DataBlade, aggregati, cast e classi operatore.

Per visualizzare gli oggetti del database:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare il database nell'elenco.

    La pagina visualizza un pannello di informazioni per ciascun oggetto del database.

  3. Per visualizzare ulteriori dettagli relativi a un oggetto del database, fare clic sul pulsante di espansione nel pannello dell'oggetto.

Visualizzazione degli oggetti della tabella

Visualizzare informazioni su colonne, indici, riferimenti, privilegi, tipi di dati definiti dall'utente (UDT), vincoli, frammenti e trigger.

Per visualizzare gli oggetti della tabella:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Per trovare una tabella, digitare qualsiasi parte del nome tabella nel campo Trova e fare clic su Cerca. È anche possibile espandere un database per visualizzare le tabelle.
  3. Facoltativo: selezionare i filtri per includere più tabelle nell'elenco:
    • Per includere i cataloghi di sistema, selezionare Include Catalogs.
    • Per includere le viste, selezionare Include Views. Una vista è una tabella virtuale basata su un'istruzione SELECT specificata.
    • Per includere i sinonimi, selezionare Include Synonyms. Un sinonimo è il nome alternativo di un oggetto di tabella, di vista o di sequenza. Generalmente, i sinonimi fanno riferimento a tabelle non presenti nel database corrente.
  4. Selezionare la tabella nell'elenco.

    La pagina visualizza un pannello di informazioni per ciascun oggetto della tabella.

  5. Per visualizzare ulteriori dettagli relativi a un oggetto della tabella, fare clic sul pulsante di espansione nel pannello dell'oggetto.

Controllo delle statistiche della tabella

Controllare le informazioni sulle statistiche per la tabella e i frammenti, inclusa l'ultima registrazione delle statistiche, la distribuzione dei dati e la percentuale delle righe modificate. Specificare le opzioni per l'aggiornamento automatico delle statistiche per la tabella.

Prerequisito:
IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.70 per specificare le opzioni per l'aggiornamento automatico delle statistiche per la tabella e per visualizzare alcune informazioni, ad esempio la percentuale di righe modificate.

Per controllare le statistiche della tabella:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Espandere il database nell'elenco.
  3. Selezionare la tabella nell'elenco.
  4. Espandere il pannello Statistiche.
  5. Per visualizzare le statistiche a livello dei frammenti, fare clic su Statistiche frammento. Queste informazioni vengono visualizzate per IBM Informix versione 11.70.

Creazione di un database

Creare un database e specificare le impostazioni internazionali del database.

Prerequisito:
Per creare un database, OpenAdmin Tool (OAT) for Informix richiede IBM Informix 11.70.xC2.


Per creare un database:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nel menu Azioni, fare clic su Crea database.
  3. Seguire le istruzioni nella procedura guidata per la creazione del database.

  • Concetti correlati:
  • Utilizzo di CREATE DATABASE (Database Design Guide)
  • Impostazioni internazionali del database (GLS User's Guide)
  • Riferimento correlato:
  • Istruzione CREATE DATABASE (SQL Syntax)
  • Argomento create database: creazione di un database (API di gestione SQL) (Administrator's Reference)

Rimozione di un database

Rimuovere un database ed eliminare le tabelle e i dati contenuti.

Prerequisito:
Per rimuovere un database, OpenAdmin Tool (OAT) for Informix richiede IBM Informix 11.70.xC2.


Per rimuovere un database:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare il database.
  3. Nel menu Azioni, fare clic su Rimuovi database.

  • Riferimento correlato:
  • Istruzione DROP DATABASE (SQL Syntax)

Gestione delle tabelle e degli indici

Creare, rimuovere e troncare le tabelle. Creare, disabilitare e rimuovere gli indici.

Creazione di una tabella

Creare una tabella e definire gli attributi necessari a renderla completamente funzionale.

Per creare una tabella:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare il database in cui creare la tabella.
  3. Nel menu Azioni, fare clic su Crea tabella.
  4. Seguire le istruzioni della procedura guidata per la creazione di tabelle.

Eliminazione di una tabella

Rimuovere una tabella con i relativi dati e indici associati.

Quando si elimina una tabella, vengono eliminati anche i dati archiviati al suo interno e gli indici, i vincoli, i trigger, i privilegi di accesso e i sinonimi locali. Vengono anche eliminate tutte le visualizzazioni basate sulla tabella e le tabelle di violazioni e diagnostica ad essa associate.

Per eliminare una tabella:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nell'elenco, espandere il database che contiene la tabella.
  3. Selezionare la tabella.
  4. Nel menu Azioni, fare clic su Elimina tabella.

Troncamento di una tabella

Eliminare tutte le righe da una tabella e liberare lo spazio di archiviazione associato o riservare lo spazio di archiviazione per poter essere riutilizzato dalla tabella e i relativi indici.

Il riutilizzo dello spazio di archiviazione può rendere più efficace la gestione dell'archiviazione in applicazioni in cui una tabella viene periodicamente svuotata e ricaricata con nuove righe.

Per troncare una tabella:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nell'elenco, espandere il database che contiene la tabella.
  3. Selezionare la tabella.
  4. Nel menu Azioni, fare clic su Tronca tabella.
  5. Facoltativo: selezionare Riserva spazio di memorizzazione affinché lo spazio di memorizzazione rimanga assegnato alla tabella.

Creazione di un indice

Creare un indice per una o più colonne in una tabella.

Per creare un indice:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nell'elenco, espandere il database e selezionare la tabella.
  3. Nel menu Azioni, fare clic su Crea indice.
  4. Seguire le istruzioni della procedura guidata per la creazione dell'indice.

  • Concetti correlati:
  • Indici e relative considerazioni sulle prestazioni (Performance Guide)
  • Riferimento correlato:
  • Istruzione CREATE INDEX (SQL Syntax)

Disabilitazione o abilitazione di un indice

Disabilitare un indice o abilitarne uno disabilitato.

Un indice è abilitato per impostazione predefinita. Il server di database riconosce l'indice e lo aggiorna durante le operazioni di inserimento, eliminazione e aggiornamento. Quando un indice viene disabilitato, non viene più aggiornato durante tali operazioni.

Per abilitare o disabilitare un indice:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nell'elenco, espandere il database che contiene la tabella alla quale si applica l'indice e selezionare la tabella.
  3. Espandere il pannello Indici.
  4. Selezionare l'indice.
  5. Nel menu Azioni del pannello, fare clic su Abilita indice o Disabilita indice.

  • Concetti correlati:
  • Modalità di trattamento degli indici disabilitati da parte del server di database (SQL Syntax)
  • Definizioni di modalità di oggetti del database (SQL Syntax)

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Nell'elenco, espandere il database che contiene la tabella alla quale si applica l'indice e selezionare la tabella.
  3. Espandere il pannello Indici.
  4. Selezionare l'indice.
  5. Nel menu Azioni del pannello, fare clic su Elimina indice.

  • Attività correlate:
  • Eliminazione di indici per una maggiore efficienza di aggiornamento tabella (Performance Guide)
  • Riferimento correlato:
  • Istruzione DROP INDEX (SQL Syntax)

Caricamento e scaricamento dei dati con tabelle esterne

Caricare e scaricare i dati gestiti e memorizzati al di fuori del server di database utilizzando le tabelle esterne.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Caricamento dei dati con una tabella esterna

Caricare dati in una tabella di database da un'origine esterna al server di database utilizzando una tabella esterna.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Per caricare dati con una tabella esterna:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare la tabella di database in cui caricare i dati.

    Per trovare una tabella, digitare qualsiasi parte del nome tabella nel campo Trova e fare clic su Cerca. È anche possibile espandere un database per visualizzare le tabelle.

  3. Nel menu Azioni, fare clic su Carica tabella.
  4. Seguire le istruzioni riportate nella procedura guidata per il caricamento delle tabelle.

Scaricamento dei dati su una tabella esterna

Scaricare i dati da una tabella di database a una tabella esterna.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Per scaricare dati su una tabella esterna:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare la tabella di database da cui scaricare i dati.

    Per trovare una tabella, digitare qualsiasi parte del nome tabella nel campo Trova e fare clic su Cerca. È anche possibile espandere un database per visualizzare le tabelle.

  3. Nel menu Azioni, fare clic su Scarica tabella.
  4. Seguire le istruzioni riportate nella procedura guidata per lo scaricamento delle tabelle.

Creazione di una tabella esterna

Creare una tabella gestita e memorizzata al di fuori del server di database per caricare e scaricare i dati.

Prerequisito:
Per creare tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6.

Per creare una tabella esterna:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Selezionare il database per il quale creare la tabella.
  3. Nel menu Azioni, fare clic su Crea tabella.
  4. Seguire le istruzioni della procedura guidata per la creazione di tabelle.

    Per Tipo tabella, selezionare un tipo di tabella esterna:

    • Esterna fissa: una tabella in formato di testo a larghezza fissa i cui dati sono gestiti e memorizzati all'esterno del server di database Informix.
    • Esterna delimitata: una tabella in formato di testo delimitato i cui dati sono gestiti e memorizzati all'esterno del server di database Informix.
    • Esterna Informix: una tabella in formato interno Informix i cui dati sono gestiti e memorizzati all'esterno del server di database Informix.

      Le tabella esterne vengono utilizzate per caricare e scaricare i dati dal database.

Visualizzazione delle informazioni della tabella esterna

Visualizzare le informazioni relative a una tabella esterna.

Queste informazioni vengono visualizzate per Informix 11.50.xC6 o versione successiva.

Per visualizzare le informazioni relative a una tabella esterna:

  1. Nel menu di IBM OpenAdmin Tool (OAT) for Informix, fare clic su SQL Toolbox > Schema Manager.
  2. Per trovare una tabella, digitare qualsiasi parte del nome tabella nel campo Trova e fare clic su Cerca. È anche possibile espandere un database per visualizzare le tabelle.
  3. Selezionare la tabella nell'elenco.

    La pagina visualizza un pannello di informazioni per ciascun oggetto della tabella.

  4. Per visualizzare ulteriori dettagli relativi a un oggetto della tabella, fare clic sul pulsante di espansione nel pannello dell'oggetto.

    I pannelli Informazioni tabella e Colonne visualizzano le informazioni specifiche delle tabelle esterne.

Visualizzazione dello stato delle attività di caricamento e scaricamento

Visualizzare lo stato delle attività inoltrate mediante IBM OpenAdmin Tool (OAT) for Informix per caricare e scaricare i dati utilizzando le tabelle esterne.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Per visualizzare lo stato delle attività di caricamento e scaricamento:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Fare clic sul database o sulla tabella su cui viene eseguita l'attività.
  3. Espandere il pannello Informazioni.
  4. Fare clic sulla scheda Stato attività.

    Per visualizzare ulteriori informazioni su un'attività, inclusi i codici di errore e i messaggi di ritorno, fare clic sull'icona dei dettagli dell'attività.

  • Concetti correlati:
  • Messaggi di errore (Administrator's Reference)

Esecuzione di attività di caricamento e scaricamento salvate

Eseguire attività che consentono il caricamento e lo scaricamento di dati mediante tabelle esterne.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Quando si caricano o si scaricano i dati utilizzando tabelle esterne, l'attività viene salvata. È possibile eseguire di nuovo l'attività o eliminarla.

Per eseguire un'attività di caricamento o scaricamento salvata:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Fare clic sul database o sulla tabella su cui viene eseguita l'attività.
  3. Espandere il pannello Informazioni.
  4. Fare clic sulla scheda Attività.
  5. Selezionare le attività da eseguire.
  6. Nel menu Azioni del pannello, fare clic su Inoltra.

Eliminazione di attività di caricamento e scaricamento

Eliminare un'attività di caricamento o scaricamento o lo stato di un'attività di caricamento o scaricamento.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Quando si caricano o si scaricano i dati utilizzando OAT, l'attività viene salvata. È possibile inoltrare di nuovo l'attività o eliminarla. È anche possibile eliminare lo stato delle attività inoltrate.

Importante:
Quando si elimina un'attività nella pagina Attività, anche i relativi risultati vengono eliminati dalla pagina Stato attività. Quando si elimina uno stato dell'attività, viene eliminato solo lo stato specificato.

Per eliminare le attività di caricamento e scaricamento:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Fare clic sul database o sulla tabella su cui viene eseguita l'attività.
  3. Espandere il pannello Informazioni.
  4. Selezionare la scheda appropriata:
    • Per eliminare l'attività e lo stato dell'attività, fare clic sulla scheda Attività.
    • Per eliminare solo lo stato dell'attività, fare clic sulla scheda Stato attività.
  5. Selezionare l'attività.
  6. Fare clic su Azioni > Elimina.

Creazione di un gruppo di attività di caricamento o scaricamento salvate

Raggruppare le attività salvate in modo da poterle inoltrare insieme.

Prerequisito:
Per caricare e scaricare dati con tabelle esterne, IBM OpenAdmin Tool (OAT) for Informix richiede IBM Informix versione 11.50.xC6 o successiva.

Quando si caricano o si scaricano i dati utilizzando OAT, l'attività viene salvata. È possibile eseguire di nuovo l'attività, eliminarla o raggrupparla insieme ad altre attività salvate.

Per creare un gruppo di attività di caricamento o scaricamento salvate:

  1. Nel menu OAT, fare clic su SQL Toolbox > Schema Manager.
  2. Fare clic sul database o sulla tabella su cui viene eseguita l'attività.
  3. Espandere il pannello Informazioni.
  4. Fare clic sulla scheda Attività.
  5. Selezionare le attività da includere nel gruppo. Le attività di caricamento e scaricamento possono essere incluse nello stesso gruppo.
  6. Nel menu Azioni del pannello, fare clic su Crea gruppo e immettere un nome per il gruppo.
