Wtyczka Menedżer schematów dla programu IBM OpenAdmin Tool for Informix

Wtyczka Menedżer schematów dla programu IBM® OpenAdmin Tool (OAT) for Informix stanowi interfejs graficzny do przeglądania baz danych i tabel oraz administrowania nimi.

Za pomocą wtyczki Menedżera schematów można wykonywać następujące zadania:

  • Tworzenie tabel i indeksów.
  • Ładowanie i wyjmowanie danych zarządzanych i zapisywanych poza serwerem bazy danych za pomocą zewnętrznych tabel.
  • Wyświetlanie informacji o obiektach bazy danych, w tym procedurach składowanych i funkcjach, sekwencjach, typach danych zdefiniowanych przez użytkownika (UDT), uprawnieniach, modułach DataBlade, agregatach, rzutowaniach i klasach operatorów.
  • Wyświetlanie informacji o tabelach, w tym kolumnach, indeksach, odwołaniach, uprawnieniach, typach zdefiniowanych przez użytkowników, ograniczeniach, fragmentach i wyzwalaczach.

Wyświetlanie obiektów bazy danych

Można wyświetlać obiekty bazy danych, w tym procedury składowane i funkcje, sekwencje, typy danych zdefiniowane przez użytkownika (UDT), uprawnienia, moduły DataBlade, agregaty, rzutowania i klasy operatorów.

Aby wyświetlać obiekty bazy danych:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Wybierz bazę danych z listy.

    Na stronie zostanie wyświetlony panel z informacjami o poszczególnych obiektach bazy danych.

  3. Aby wyświetlić szczegółowe informacje o obiekcie, kliknij przycisk rozwijania na panelu obiektu.

Wyświetlanie obiektów tabeli

Możliwe jest wyświetlanie informacji o kolumnach, indeksach, odwołaniach, uprawnieniach, typach zdefiniowanych przez użytkownika, ograniczeniach, fragmentach i wyzwalaczach.

Aby wyświetlać obiekty tabeli:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Aby znaleźć tabelę, rozpocznij wpisywanie dowolnej części jej nazwy w polu Znajdź, co spowoduje wyświetlenie pasujących tabel. Można także rozwinąć bazę danych, aby wyświetlić tabele.
  3. Opcjonalne: Wybierz filtry, aby uwzględnić dodatkowe tabele na liście:
    • Aby uwzględnić katalogi systemowe, wybierz opcję Uwzględnij katalogi.
    • Aby uwzględnić widoki, wybierz opcję Uwzględnij widoki. Widok jest tabelą wirtualną o zawartości wynikającej z wykonania określonej instrukcji SELECT.
    • Aby uwzględnić synonimy, wybierz opcję Uwzględnij synonimy. Synonim jest alternatywną nazwą obiektu tabeli, widoku lub sekwencji. Zwykle synonim odwołuje się do tabel, które nie znajdują się w bieżącej bazie danych.
  4. Wybierz tabelę na liście.

    Na stronie zostanie wyświetlony panel z informacjami o poszczególnych obiektach tabeli.

  5. Aby wyświetlić szczegółowe informacje o obiekcie tabeli, kliknij przycisk rozwijania na panelu obiektu.

Przeglądanie statystyk dot. tabel i fragmentów

Przeglądanie informacji o tabelach i fragmentach, w tym wartości procentowe zmienionych wierszy oraz liczba operacji UPDATE, DELETE i INSERT. Sprawdź, kiedy statystyki zostały zarejestrowane po raz ostatni i czas wymagany do obliczenia statystyk.

Wymaganie wstępne:
Aby wyświetlić niektóre informacje, w tym wartość procentową zmienionych wierszy, liczbę operacji i czas kompilacji potrzebny do obliczenia statystyk, program IBM OpenAdmin Tool (OAT) for Informix wymaga produktu IBM Informix w wersji 11.70.

Aby przeglądać statystyki tabel i fragmentów:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Rozwiń bazę danych na liście.
  3. Wybierz tabelę na liście.
  4. Rozwiń panel Statystyki.
  5. Aby wyświetlić statystyki na poziomie fragmentu, kliknij opcję Statystyki fragmentów. Ta informacja jest wyświetlana dla produktu IBM Informix, wersja 11.70.

Zarządzanie tabelami i indeksami

Tworzenie, usuwanie i obcinanie tabel. Tworzenie, wyłączanie i usuwanie indeksów.

Tworzenie tabel

Tworzenie tabeli i zdefiniowanie atrybutów potrzebnych, aby była w pełni funkcjonalna.

Aby utworzyć tabelę:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Wybierz bazę danych, w której chcesz utworzyć tabelę.
  3. W menu Działania kliknij opcję Utwórz tabelę.
  4. Postępuj zgodnie z instrukcjami wyświetlanymi przez Kreatora tabeli.

Usuwanie tabeli

Usuwanie tabeli z powiązanymi indeksami i danymi.

Usunięcie tabeli powoduje również usunięcie danych zapisanych w jej indeksach, ograniczeniach, wyzwalaczach, uprawnieniach dostępu i lokalnych synonimach. Powoduje również usunięcie wszystkich widoków opartych na tabeli oraz wszelkich naruszeń i tabel diagnostycznych skojarzonych z tabelą.

W celu usunięcia tabeli:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Na liście rozwiń bazę danych, która zawiera tabelę.
  3. Wybierz tabelę.
  4. W menu Działania kliknij opcję Usuń tabelę.

Obcinanie tabel

Usuwa wszystkie wiersze z tabeli i zwalnia pamięć masową lub rezerwuje miejsce w pamięci masowej do ponownego wykorzystania przez tabelę i jej indeksy.

Ponowne wykorzystanie pamięci masowej może przyczynić się do zwiększenia wydajności zarządzania pamięcią masową w aplikacjach, w których tabela jest okresowo opróżniania i do której ładowane są nowe wiersze.

Aby obciąć tabelę:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Na liście rozwiń bazę danych, która zawiera tabelę.
  3. Wybierz tabelę.
  4. W menu Działania kliknij opcję Obetnij tabelę.
  5. Opcjonalne: Wybierz opcję Zarezerwuj obszar pamięci masowej, aby zachować pamięć masową przydzieloną do tabeli.

Tworzenie indeksu

Tworzenie indeksu dla co najmniej jednej kolumny w tabeli.

Aby utworzyć indeks:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Na liście rozwiń bazę danych i wybierz tabelę.
  3. W menu Działania kliknij opcję Utwórz indeks.
  4. Postępuj zgodnie z instrukcjami wyświetlanymi przez kreator Utwórz indeks.

Wyłączanie/włączanie indeksu

Wyłączanie indeksu oraz włączanie wyłączonego indeksu.

Indeks jest domyślnie włączony. Serwer bazy danych rozpoznaje indeks i aktualizuje go podczas wykonywania operacji wstawiania, usuwania i aktualizowania. Jeśli indeks jest wyłączony, nie jest aktualizowany w czasie wykonywania tych operacji.

W celu włączenia lub wyłączenia indeksu:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Na liście rozwiń bazę danych zawierającą tabelę, której dotyczy indeks, i wybierz tabelę.
  3. Rozwiń panel Indeksy.
  4. Wybierz indeks.
  5. W menu Działania panelu kliknij opcję Włącz indeks lub Wyłącz indeks.

  • Informacje pokrewne
  • Jak serwer bazy danych traktuje wyłączone indeksy
  • Definicje trybów obiektu bazy danych

Usuwanie indeksu

Usuwanie indeksu; na przykład: usuwanie niewydajnego indeksu.

W celu usunięcia indeksu:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Na liście rozwiń bazę danych zawierającą tabelę, której dotyczy indeks, i wybierz tabelę.
  3. Rozwiń panel Indeksy.
  4. Wybierz indeks.
  5. W menu Działania panelu kliknij opcję Usuń indeks.

  • Informacje pokrewne
  • Usuwanie indeksów w celu zwiększenia efektywności aktualizowania tabel
  • Instrukcja DROP INDEX

Ładowanie i wyjmowanie danych z zewnętrznych tabel

Ładowanie i wyjmowanie danych zarządzanych i zapisywanych poza serwerem bazy danych za pomocą zewnętrznych tabel.

Ładowanie danych za pomocą tabeli zewnętrznej

Ładowanie danych do tabeli bazy danych ze źródła, które jest zewnętrzne względem serwera bazy danych, za pomocą tabeli zewnętrznej.

Aby załadować dane za pomocą tabeli zewnętrznej:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Wybierz tabelę bazy danych, do której zostaną załadowane dane.

    Aby znaleźć tabelę, rozpocznij wpisywanie dowolnej części jej nazwy w polu Znajdź, co spowoduje wyświetlenie pasujących tabel. Można także rozwinąć bazę danych, aby wyświetlić tabele.

  3. W menu Działania kliknij opcję Załaduj tabelę.
  4. Postępuj zgodnie z instrukcjami wyświetlanymi przez kreatora ładowania do tabeli.

Usuwanie danych do tabeli zewnętrznej

Usuwanie danych z tabeli bazy danych do tabeli zewnętrznej.

Aby usunąć dane do tabeli zewnętrznej:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Wybierz tabelę bazy danych, z której dane zostaną usunięte.

    Aby znaleźć tabelę, rozpocznij wpisywanie dowolnej części jej nazwy w polu Znajdź, co spowoduje wyświetlenie pasujących tabel. Można także rozwinąć bazę danych, aby wyświetlić tabele.

  3. W menu Działania kliknij opcję Usuń tabelę z pamięci.
  4. Postępuj zgodnie z instrukcjami wyświetlanymi przez kreatora usuwania tabeli z pamięci.

Tworzenie tabeli zewnętrznej

Tworzenie tabeli służące do ładowania i wyjmowania danych zarządzanych i zapisywanych poza serwerem bazy danych.

Wymaganie wstępne:
W celu utworzenia tabel zewnętrznych program wymaga produktu IBM Informix w wersji 11.50.xC6.

Aby utworzyć tabelę zewnętrzną:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Wybierz bazę danych, dla której ma zostać utworzona tabela.
  3. W menu Działania kliknij opcję Utwórz tabelę.
  4. Postępuj zgodnie z instrukcjami wyświetlanymi przez Kreatora tabeli.

    Dla pola Typ tabeli wybierz typ tabeli zewnętrznej:

    • Zewnętrzna-stała: Tabela w formacie tekstu o stałej szerokości, której dane są zarządzane i zapisywane poza serwerem bazy danych Informix.
    • Zewnętrzna-rozdzielona znakiem: Tabela w formacie tekstu rozdzielone znakiem, której dane są zarządzane i zapisywane poza serwerem bazy danych Informix.
    • Zewnętrzna-Informix: Tabela w formacie wewnętrznym Informix, której dane są zarządzane i zapisywane poza serwerem bazy danych Informix.

      Tabele zewnętrzne służą do ładowania i wyjmowania danych z bazy danych.

Wyświetlanie informacji o tabelach zewnętrznych

Wyświetlanie informacji o tabelach zewnętrznych.

Aby wyświetlić informacje o tabeli zewnętrznej:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Aby znaleźć tabelę, rozpocznij wpisywanie dowolnej części jej nazwy w polu Znajdź, co spowoduje wyświetlenie pasujących tabel. Można także rozwinąć bazę danych, aby wyświetlić tabele.
  3. Wybierz tabelę na liście.

    Na stronie zostanie wyświetlony panel z informacjami o poszczególnych obiektach tabeli.

  4. Aby wyświetlić szczegółowe informacje o obiekcie tabeli, kliknij przycisk rozwijania na panelu obiektu.

    Na panelu Informacje o tabeli i panelu Kolumny są wyświetlane informacje specyficzne dla danych tabel zewnętrznych.

Wyświetlanie statusu zadań ładowania i wyjmowania

Wyświetlanie statusu zadań ładowania i wyjmowania danych za pomocą zewnętrznych tabel.

Aby wyświetlić status zadania ładowania i wyjmowania:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Kliknij bazę danych lub tabelę, dla której wykonywane jest zadanie.
  3. Rozwiń panel Informacje.
  4. Kliknij kartę Status zadania.
  5. Z listy Pokaż wybierz typ statusu zadania do wyświetlenia: status wszystkich zadań, zadań ładowania lub zadań wyjmowania.

    Aby wyświetlić więcej informacji na temat zadania, w tym kody błędów i zwrócone komunikaty, kliknij przycisk szczegółów zadania.

  • Informacje pokrewne
  • Komunikaty o błędach

Uruchamianie zadań ładowania i wyjmowania

Uruchamianie zadań ładowania i wyjmowania danych za pomocą zewnętrznych tabel.

Podczas ładowania lub wyjmowania danych za pomocą zewnętrznych tabel, zadanie to jest zapisywane. Zadanie można uruchomić ponownie lub je usunąć.

Aby uruchomić zapisane zadanie ładowania lub wyjmowania:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Kliknij bazę danych lub tabelę, dla której wykonywane jest zadanie.
  3. Rozwiń panel Informacje.
  4. Kliknij kartę Status zadania.
  5. Z listy Pokaż wybierz typ zadań do wyświetlenia: Wszystkie zadania, Zadania ładowania lub Zadania wyjmowania.
  6. Wybierz zadania przeznaczone do uruchomienia.
  7. W menu Działania panelu kliknij opcję Wprowadź.

Usuwanie zadań ładowania lub wyjmowania

Usuwanie zadań ładowania lub wyjmowania albo statusu zadania ładowania lub wyjmowania.

Podczas ładowania lub wyjmowania danych za pomocą zewnętrznych tabel, zadanie to jest zapisywane. Zadanie można wprowadzić ponownie lub je usunąć.

Po usunięciu zadania ładowania lub wyjmowania wszystkie opisy statusów zostaną również usunięte. Ale po usunięciu statusu zadanie nie jest usuwane.

Aby usunąć zadania ładowania i wyjmowania:

  1. W menu programu IBM OpenAdmin Tool (OAT) for Informix kliknij element Narzędzia SQL → Menedżer schematów.
  2. Kliknij bazę danych lub tabelę, dla której wykonywane jest zadanie.
  3. Rozwiń panel Informacje.
  4. Kliknij kartę Status zadania.
  5. Z listy Pokaż wybierz typ zadania do usunięcia.
    • Aby usunąć tylko status, wybierz opcje Status wszystkich zadań, Status zadań ładowania lub Status zadań wyjmowania.
    • Aby usunąć zadanie i status, wybierz opcję Wszystkie zadania, Zadania ładowania lub Zadania wyjmowania.
  6. Wybierz zadania przeznaczone do usunięcia.
  7. W menu Działania panelu kliknij opcję Usuń.

Tworzenie grupy zapisanych zadań ładowania i wyjmowania

Grupowanie zapisanych zadań w celu wprowadzania ich razem.

Podczas ładowania lub wyjmowania danych za pomocą zewnętrznych tabel, zadanie to jest zapisywane. Takie zadanie można uruchomić ponownie, usunąć je lub zgrupować z innymi zapisanymi zadaniami.

Aby utworzyć grupę zapisanych zadań ładowania i wyjmowania:

  1. W menu programu OAT kliknij element Narzędzia SQL → Menedżer schematów.
  2. Kliknij bazę danych lub tabelę, dla której wykonywane jest zadanie.
  3. Rozwiń panel Informacje.
  4. Kliknij kartę Status zadania.
  5. Z listy Pokaż wybierz typ zadań do wyświetlenia: Wszystkie zadania, Zadania ładowania lub Zadania wyjmowania.
  6. Wybierz zadania, które mają zostać uwzględnione w grupie. Zadania ładowania lub wyjmowania można uwzględniać w tej samej grupie.
  7. W menu Działania panelu kliknij opcję Utwórz grupę i wprowadź nazwę grupy.

Grupy zapisanych zadań ładowania i wyjmowania dla serwera zostaną wyświetlone na stronie Status zadania bazy danych sysadmin. Aby wyświetlić grupy wybierz z listy baz danych sysadmin i rozwiń panel Informacje, a następnie kliknij kartę Status zadania.
