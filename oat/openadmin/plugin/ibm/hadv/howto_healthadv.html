Informix Health Advisor Plug-in for OAT

The IBM® Informix® Health Advisor Plug-in for OpenAdmin Tool (OAT) analyzes the state of the Informix database server. The Health Advisor plug-in gathers information about the database server, the databases, and the operating system, and creates a report that contains the results and recommendations.

Prerequisite:
The Health Advisor plug-in requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in includes a series of alarms that check conditions on the database server: configuration, Enterprise Replication (ER), operating system (OS), performance, and storage. You can create profiles that specify the alarms that are enabled and the thresholds for the alarms. You can schedule a task to run the Health Advisor with a specific profile at regular intervals. You can specify who receives an email notification of the results. You can also run the Health Advisor on demand on the current profile and view the report.

Running the Health Advisor

Run the Health Advisor on demand on the current profile and view the report.

Prerequisite:
The IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in analyzes the state of the Informix database server. A Health Advisor profile specifies the alarms that are enabled and the thresholds for the alarms.

To run the Health Advisor:

  1. On the OAT menu, click Health Center > Health Advisor.
  2. Optional: To change the current profile, on the Profile page, click the profile in the list, and then click Load.
  3. Click Run Health Advisor. The report is displayed in a new browser window.

Adding a profile

Add a profile to the IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT).

Prerequisite:
The Health Advisor plug-in requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in analyzes the state of the Informix database server. You can create profiles that specify the alarms that are enabled and the thresholds for the alarms. You can schedule a task to run the Health Advisor with a specific profile at regular intervals.

To add a profile:

  1. On the OAT menu, click Health Center > Health Advisor.
  2. On the Profile page, enter a unique name for the profile and click Add.

Setting up alarms for a profile

Specify the alarms that are enabled for a profile and the thresholds for the alarms in the IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT).

Prerequisite:
The Health Advisor plug-in requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in analyzes the state of the Informix database server. A Health Advisor profile specifies the alarms that are enabled and the thresholds for the alarms.

To set up the alarms for a profile:

  1. On the OAT menu, click Health Center > Health Advisor.
  2. Optional: To change the current profile, on the Profile page, click the profile in the list, and then click Load.
  3. Click the Alarms tab.
  4. Select the Enable check box for the alarms to include in the profile.
  5. Click Modify Threshold to review or change the thresholds for an alarm.

Scheduling a task for a profile

Schedule a task to run the Health Advisor with a specific profile at regular intervals.

Prerequisite:
The IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in analyzes the state of the Informix database server. A Health Advisor profile specifies the alarms that are enabled and the thresholds for the alarms. You can schedule a task to run the Health Advisor for the current profile.

To schedule a task for the current profile:

  1. On the OAT menu, click Health Center > Health Advisor.
  2. Optional: To change the current profile, on the Profile page, click the profile in the list, and then click Load.
  3. Click the Schedule tab.
  4. Specify the schedule for the task by using the configuration fields.

Configuring email notification

Configure the email program and specify the recipients of the results of a check.

Prerequisite:
The IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requires IBM Informix 11.50.xC7 or later.

The Health Advisor plug-in creates a report that contains the results of a check and the recommendations. The email configuration is based on the operating system that sends the email. The email is sent from the computer where the database server is located. You can configure the email program for the current profile.

To configure email notification for the current profile:

  1. On the OAT menu, click Health Center > Health Advisor.
  2. Optional: To change the current profile, on the Profile page, click the profile in the list, and then click Load.
  3. Click the Notification tab. For examples of the configuration, click the help button for the page.
