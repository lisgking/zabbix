Informix-Plug-in 'Health Advisor' für OAT

Das IBM® Informix®-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) analysiert den Status des Informix-Datenbankservers. Das Plug-in 'Health Advisor' stellt Informationen zum Datenbankserver, zu den Datenbanken und dem Betriebssystem zusammen und erstellt einen Bericht mit den Ergebnissen und Empfehlungen.

Voraussetzung:
Das Plug-in 'Health Advisor' erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' enthält eine Reihe von Alarmen zur Prüfung der Betriebsbedingungen auf dem Datenbankserver: Konfiguration, Enterprise Replication (ER), Betriebssystem, Leistung und Speicher. Sie können Profile erstellen, die die aktivierten Alarme und die für die Alarme definierten Schwellenwerte angeben. Sie können eine Task terminieren, um Health Advisor in regelmäßigen Abständen mit einem bestimmten Profil auszuführen. Sie können angeben, wer eine E-Mail-Benachrichtigung mit den Ergebnissen erhalten soll. Sie können Health Advisor auch bei Bedarf mit dem aktuellen Profil ausführen und den Bericht anzeigen.

Ausführen von Health Advisor

Führen Sie Health Advisor bei Bedarf mit dem aktuellen Profil aus und zeigen Sie den Bericht an.

Voraussetzung:
Das IBM Informix-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' analysiert den Status des Informix-Datenbankservers. Ein Health Advisor-Profil gibt die aktivierten Alarme und die für die Alarme definierten Schwellenwerte an.

Gehen Sie wie folgt vor, um Health Advisor auszuführen:

  1. Klicken Sie im OAT-Menü auf Diagnosezentrale > Health Advisor.
  2. Optional: Klicken Sie in der Liste auf der Seite Profil auf das gewünschte Profil und anschließen auf Laden, um das aktuelle Profil zu ändern.
  3. Klicken Sie auf Health Advisor ausführen. Der Bericht wird in einem neuen Browserfenster angezeigt.

Hinzufügen eines Profils

Sie können dem IBM Informix-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) ein Profil hinzufügen.

Voraussetzung:
Das Plug-in 'Health Advisor' erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' analysiert den Status des Informix-Datenbankservers. Sie können Profile erstellen, die die aktivierten Alarme und die für die Alarme definierten Schwellenwerte angeben. Sie können eine Task terminieren, um Health Advisor in regelmäßigen Abständen mit einem bestimmten Profil auszuführen.

Gehen Sie wie folgt vor, um ein Profil hinzuzufügen:

  1. Klicken Sie im OAT-Menü auf Diagnosezentrale > Health Advisor.
  2. Geben Sie auf der Seite Profil einen eindeutigen Namen für das Profil ein und klicken Sie auf Hinzufügen.

Konfigurieren von Alarmen für ein Profil

Geben Sie die für ein Profil aktivierten Alarme und die für die Alarme definierten Schwellenwerte im IBM Informix-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) an.

Voraussetzung:
Das Plug-in 'Health Advisor' erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' analysiert den Status des Informix-Datenbankservers. Ein Health Advisor-Profil gibt die aktivierten Alarme und die für die Alarme definierten Schwellenwerte an.

Gehen Sie wie folgt vor, um die Alarme für ein Profil zu konfigurieren:

  1. Klicken Sie im OAT-Menü auf Diagnosezentrale > Health Advisor.
  2. Optional: Klicken Sie in der Liste auf der Seite Profil auf das gewünschte Profil und anschließen auf Laden, um das aktuelle Profil zu ändern.
  3. Klicken Sie auf die Registerkarte Alarme.
  4. Wählen Sie das Kontrollkästchen Aktivieren für die Alarme aus, die in das Profil eingeschlossen werden sollen.
  5. Klicken Sie auf Schwellenwerte modifizieren, um die Schwellenwerte für einen Alarm anzuzeigen oder zu ändern.

Terminieren einer Task für ein Profil

Terminieren Sie eine Task, Health Advisor in regelmäßigen Abständen mit einem bestimmten Profil auszuführen.

Voraussetzung:
Das IBM Informix-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' analysiert den Status des Informix-Datenbankservers. Ein Health Advisor-Profil gibt die aktivierten Alarme und die für die Alarme definierten Schwellenwerte an. Sie können eine Task terminieren, um Health Advisor mit dem aktuellen Profil auszuführen.

Gehen Sie wie folgt vor, um eine Task für das aktuelle Profil zu terminieren:

  1. Klicken Sie im OAT-Menü auf Diagnosezentrale > Health Advisor.
  2. Optional: Klicken Sie in der Liste auf der Seite Profil auf das gewünschte Profil und anschließen auf Laden, um das aktuelle Profil zu ändern.
  3. Klicken Sie auf die Registerkarte Zeitplan.
  4. Geben Sie den Zeitplan für die Task mithilfe der Konfigurationsfelder an.

Konfigurieren von E-Mail-Benachrichtigungen

Konfigurieren Sie das E-Mail-Programm und geben Sie die Empfänger für die Ergebnisse einer Prüfung an.

Voraussetzung:
Das IBM Informix-Plug-in 'Health Advisor' für OpenAdmin Tool (OAT) erfordert IBM Informix ab Version 11.50.xC7.

Das Plug-in 'Health Advisor' erstellt einen Bericht mit den Ergebnissen einer Prüfung und den Empfehlungen. Die E-Mail-Konfiguration basiert auf dem Betriebssystem, über das die E-Mail gesendet wird. Die E-Mail wird über den Computer gesendet, auf dem sich der Datenbankserver befindet. Sie können das E-Mail-Programm für das aktuelle Profil konfigurieren.

Gehen Sie wie folgt vor, um die E-Mail-Benachrichtigung für das aktuelle Profil zu konfigurieren:

  1. Klicken Sie im OAT-Menü auf Diagnosezentrale > Health Advisor.
  2. Optional: Klicken Sie in der Liste auf der Seite Profil auf das gewünschte Profil und anschließen auf Laden, um das aktuelle Profil zu ändern.
  3. Klicken Sie auf die Registerkarte Benachrichtigung. Sie können durch Klicken auf die Schaltfläche Hilfe für die Seite Konfigurationsbeispiele aufrufen.
