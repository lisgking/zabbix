Подключаемый модуль советчика по работоспособности Informix для OAT

Подключаемый модуль IBM® Informix® Health Advisor для OpenAdmin Tool (OAT) анализирует состояние сервера баз данных Informix. Подключаемый модуль Health Advisor собирает информацию о сервере баз данных, базах данных и операционной системе, а также создает отчет, содержащий результаты и рекомендации.

Предварительное требование:
Для подключаемого модуля Health Advisor требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor включает в себя ряд оповещений, которые проверяют условия на сервере баз данных: конфигурацию, Enterprise Replication (ER), операционную систему, производительность и систему хранения. Вы можете создать профиль, который задает включенные оповещения и пороги для этих оповещений. Можно составить расписание для задачи запуска монитора работоспособности с определенным профилем через регулярные интервалы. Здесь же можно указать, кто будет получать уведомления по электронной почте о результатах проверок. Можно также запустить советчик по работоспособности по требованию с текущим профилем и посмотреть его отчет.

Запуск советчика по работоспособности

Запустите советчик по работоспособности по требованию с текущим профилем и посмотрите его отчет.

Предварительное требование:
Для подключаемого модуля IBM Informix Health Advisor для OpenAdmin Tool (OAT) требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor анализирует состояние сервера баз данных Informix. Профиль советчика по работоспособности задает включенные оповещения и пороги для этих оповещений.

Чтобы запустить советчик по работоспособности:

  1. В меню OAT выберите Центр работоспособности > Советчик по работоспособности.
  2. Необязательно: Чтобы изменить текущий профиль, на странице Профиль щелкните по профилю в списке и выберите Загрузить.
  3. Выберите Запустить советчик по работоспособности. Отчет будет выведен в новом окне браузера.

Добавление профиля

Добавьте профиль в подключаемый модуль IBM Informix Health Advisor для OpenAdmin Tool (OAT).

Предварительное требование:
Для подключаемого модуля Health Advisor требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor анализирует состояние сервера баз данных Informix. Вы можете создать профиль, который задает включенные оповещения и пороги для этих оповещений. Можно составить расписание для задачи запуска монитора работоспособности с определенным профилем через регулярные интервалы.

Чтобы добавить профиль:

  1. В меню OAT выберите Центр работоспособности > Советчик по работоспособности.
  2. На странице Профиль введите уникальное имя профиля и нажмите кнопку Добавить.

Задание оповещений для профиля

Задайте включенные оповещения и пороги для этих оповещений в подключаемом модуле IBM Informix Health Advisor для OpenAdmin Tool (OAT).

Предварительное требование:
Для подключаемого модуля Health Advisor требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor анализирует состояние сервера баз данных Informix. Профиль советчика по работоспособности задает включенные оповещения и пороги для этих оповещений.

Чтобы задать оповещения для профиля:

  1. В меню OAT выберите Центр работоспособности > Советчик по работоспособности.
  2. Необязательно: Чтобы изменить текущий профиль, на странице Профиль щелкните по профилю в списке и выберите Загрузить.
  3. Откройте вкладку Оповещения.
  4. Выберите Включить для оповещений, чтобы включить их в профиль.
  5. Выберите Изменить порог, чтобы посмотреть или изменить пороги для оповещения.

Планирование задачи для профиля

Составьте расписание для задачи запуска монитора работоспособности с определенным профилем через регулярные интервалы.

Предварительное требование:
Для подключаемого модуля IBM Informix Health Advisor для OpenAdmin Tool (OAT) требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor анализирует состояние сервера баз данных Informix. Профиль советчика по работоспособности задает включенные оповещения и пороги для этих оповещений. Можно составить расписание задачи, чтобы запускать монитор работоспособности с текущим профилем.

Чтобы включить в расписание задачу для текущего профиля:

  1. В меню OAT выберите Центр работоспособности > Советчик по работоспособности.
  2. Необязательно: Чтобы изменить текущий профиль, на странице Профиль щелкните по профилю в списке и выберите Загрузить.
  3. Откройте вкладку Расписание.
  4. Задайте расписание для задачи, используя поля конфигурирования.

Конфигурирование уведомлений по электронной почте

Сконфигурируйте программу электронной почты и задайте получателей уведомлений по электронной почте.

Предварительное требование:
Для подключаемого модуля IBM Informix Health Advisor для OpenAdmin Tool (OAT) требуется IBM Informix 11.50.xC7 или новее.

Подключаемый модуль Health Advisor создает отчет, содержащий результаты проверки и рекомендации. Конфигурация электронной почты основана на операционной системе, которая отправляет сообщение электронной почты. Электронная почта отправляется с компьютера, где расположен сервер баз данных. Можно сконфигурировать программу электронной почты для текущего профиля.

Чтобы сконфигурировать уведомления электронной почты для текущего профиля:

  1. В меню OAT выберите Центр работоспособности > Советчик по работоспособности.
  2. Необязательно: Чтобы изменить текущий профиль, на странице Профиль щелкните по профилю в списке и выберите Загрузить.
  3. Откройте вкладку Уведомление. Чтобы посмотреть примеры конфигураций, нажмите кнопку справки на странице.
