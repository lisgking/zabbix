Plug-in Informix Health Advisor para OAT

El plug-in de IBM® Informix® Health Advisor para la herramienta OpenAdmin (OAT) analiza el estado del servidor de bases de datos de Informix. El plug-in Health Advisor recopila información sobre el servidor de bases de datos, las bases de datos y el sistema operativo y crea un informe que contiene resultados y recomendaciones.

Requisito previo:
El plug-in Health Advisor requiere IBM Informix 11.50.xC7 o posterior.

El plug-in Health Advisor incluye una serie de alarmar que comprueban la condiciones en el servidor de bases de datos: configuración, Enterprise Replication (ER), sistema operativo (SO), rendimiento y almacenamiento. Puede crear perfiles que especifiquen las alarmas habilitadas y los umbrales de las alarmas. Puede planificar una tarea para ejecutar Health Advisor con un perfil específico a intervalos regulares. Puede especificar quién va a recibir una notificación de correo electrónico de los resultados. También puede ejecutar Health Advisor a petición en el perfil actual y ver el informe.

Ejecución de Health Advisor

Ejecute Health Advisor a petición en el perfil actual y visualice el informe.

Requisito previo:
El plug-in IBM Informix Health Advisor para la herramienta OpenAdmin (OAT) requiere IBM Informix 11.50.xC7 o posterior.

El plug-in de Health Advisor analiza el estado del servidor de bases de datos de Informix. Un perfil de Health Advisor especifica las alarmas habilitadas y los umbrales de estas alarmas.

Para ejecutar Health Advisor:

  1. En el menú de OAT, pulse Centro de salud > Health Advisor.
  2. Opcional: para cambiar el perfil actual, en la página Perfil, pulse el perfil de la lista y, a continuación, pulse Cargar.
  3. Pulse Ejecutar Health Advisor.El informe se muestra en una ventana de navegador.

Adición de un perfil

Añada un perfil al plug-in IBM Informix Health Advisor para la herramienta OpenAdmin (OAT).

Requisito previo:
El plug-in Health Advisor requiere IBM Informix 11.50.xC7 o posterior.

El plug-in de Health Advisor analiza el estado del servidor de bases de datos de Informix. Puede crear perfiles que especifiquen las alarmas habilitadas y los umbrales de las alarmas. Puede planificar una tarea para ejecutar Health Advisor con un perfil específico a intervalos regulares.

Para añadir un perfil:

  1. En el menú de OAT, pulse Centro de salud > Health Advisor.
  2. En la página Perfil, escriba un nombre exclusivo del perfil y pulse Añadir.

Configuración de alarmas para un perfil

Especifique las alarmas habilitadas para un perfil y los umbrales de dichas alarmas en el plug-in IBM Informix Health Advisor para la herramienta OpenAdmin (OAT).

Requisito previo:
El plug-in Health Advisor requiere IBM Informix 11.50.xC7 o posterior.

El plug-in de Health Advisor analiza el estado del servidor de bases de datos de Informix. Un perfil de Health Advisor especifica las alarmas habilitadas y los umbrales de estas alarmas.

Para configurar las alarmas de un perfil:

  1. En el menú de OAT, pulse Centro de salud > Health Advisor.
  2. Opcional: para cambiar el perfil actual, en la página Perfil, pulse el perfil de la lista y, a continuación, pulse Cargar.
  3. Pulse el separador Alarmas.
  4. Marque el recuadro de selección Habilitar de las alarmas que desea incluir en el perfil.
  5. Pulse Modificar umbrales para revisar o cambiar los umbrales de una alarma.

Planificación de una tarea para un perfil

Planifique una tarea de modo que ejecute Health Advisor con un perfil específico a intervalos regulares.

Requisito previo:
El plug-in IBM Informix Health Advisor para la herramienta OpenAdmin (OAT) requiere IBM Informix 11.50.xC7 o posterior.

El plug-in de Health Advisor analiza el estado del servidor de bases de datos de Informix. Un perfil de Health Advisor especifica las alarmas habilitadas y los umbrales de estas alarmas. Puede planificar una tarea para que ejecute Health Advisor para el perfil actual.

Para planificar una tarea en el perfil actual:

  1. En el menú de OAT, pulse Centro de salud > Health Advisor.
  2. Opcional: para cambiar el perfil actual, en la página Perfil, pulse el perfil de la lista y, a continuación, pulse Cargar.
  3. Pulse el separador Planificar.
  4. Especifique la planificación de la tarea mediante los campos de configuración.

Configuración de la notificación de correo electrónico

Configure el programa de correo electrónico y especifique los destinatarios del resultado de una comprobación.

Requisito previo:
El plug-in IBM Informix Health Advisor para la herramienta OpenAdmin (OAT) requiere IBM Informix 11.50.xC7 o posterior.

El plug-in Health Advisor crea un informe que contiene los resultados de una comprobación y las recomendaciones. La configuración del correo electrónico se basa en el sistema operativo que envía el mensaje de correo electrónico. El correo electrónico se envía desde el sistema donde el servidor de bases de datos está ubicado. Puede configurar el programa de correo electrónico para el perfil actual.

Para configurar la notificación de correo electrónico para el perfil actual:

  1. En el menú de OAT, pulse Centro de salud > Health Advisor.
  2. Opcional: para cambiar el perfil actual, en la página Perfil, pulse el perfil de la lista y, a continuación, pulse Cargar.
  3. Pulse el separador Notificación.Para ver ejemplos de la configuración, pulse el botón de ayuda correspondiente a la página.
