Informix Health Advisor Plug-in for OAT

O IBM® Informix® Health Advisor Plug-in for OpenAdmin Tool (OAT) analisa o estado do servidor de banco de dados Informix. O Health Advisor plug-in reúne informações sobre o servidor de banco de dados, sobre os bancos de dados, sobre o sistema operacional e cria um relatório contendo os resultados e as recomendações.

Pré-requisito:
O Health Advisor plug-in requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in inclui vários alarmes que verificam as condições sobre o servidor de banco de dados: configuração, Enterprise Replication (ER), sistema operacional (O.S.), desempenho e armazenamento. É possível criar perfis que especificam os alarmes que serão ativados e os limites para eles. É possível planejar uma tarefa para executar o Health Advisor com um perfil específico em intervalos regulares. É possível especificar quem receberá uma notificação por email dos resultados. Também é possível executar o Health Advisor on demand no perfil atual e visualizar o relatório.

Executando o Health Advisor

Execute o Health Advisor on demand no perfil atual e visualize o relatório.

Pré-requisito:
O IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in analisa o estado do servidor de banco de dados Informix. Um perfil do Health Advisor especifica os alarmes que serão ativados e os limites para eles.

Para executar o Health Advisor:

  1. No menu OAT, clique em Health Center > Health Advisor.
  2. Opcional: Para alterar o perfil atual, na página Perfil, clique no perfil na lista e, em seguida, clique em Carregar.
  3. Clique em Executar o Health Advisor. O relatório é exibido em uma nova janela do navegador.

Incluindo um Perfil

Inclua um perfil no IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT).

Pré-requisito:
O Health Advisor plug-in requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in analisa o estado do servidor de banco de dados Informix. É possível criar perfis que especificam os alarmes que serão ativados e os limites para eles. É possível planejar uma tarefa para executar o Health Advisor com um perfil específico em intervalos regulares.

Para incluir um perfil:

  1. No menu OAT, clique em Health Center > Health Advisor.
  2. Na página Perfil, insira um nome exclusivo para o perfil e clique em Incluir.

Configurando Alarmes para um Perfil

Especifique os alarmes que serão ativados para um perfil e os limites para eles no IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT).

Pré-requisito:
O Health Advisor plug-in requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in analisa o estado do servidor de banco de dados Informix. Um perfil do Health Advisor especifica os alarmes que serão ativados e os limites para eles.

Para configurar os alarmes para um perfil:

  1. No menu OAT, clique em Health Center > Health Advisor.
  2. Opcional: Para alterar o perfil atual, na página Perfil, clique no perfil na lista e, em seguida, clique em Carregar.
  3. Clique na guia Alarmes.
  4. Marque a caixa de seleção Ativar para os alarmes a serem incluídos no perfil.
  5. Clique em Modificar Limite para revisar ou alterar os limites para um alarme.

Planejando uma Tarefa para um Perfil

Planeje uma tarefa no Health Advisor com um perfil específico em intervalos regulares.

Pré-requisito:
O IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in analisa o estado do servidor de banco de dados Informix. Um perfil do Health Advisor especifica os alarmes que serão ativados e os limites para eles. É possível planejar uma tarefa para executar o Health Advisor para o perfil atual.

Para planejar uma tarefa para o perfil atual:

  1. No menu OAT, clique em Health Center > Health Advisor.
  2. Opcional: Para alterar o perfil atual, na página Perfil, clique no perfil na lista e, em seguida, clique em Carregar.
  3. Clique na guia Planejamento.
  4. Especifique o planejamento para a tarefa usando os campos de configuração.

Configurando Notificação por E-mail

Configure o programa de e-mail e especifique os destinatários dos resultados de uma verificação.

Pré-requisito:
O IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT) requer o IBM Informix 11.50.xC7 ou mais recente.

O Health Advisor plug-in cria um relatório contendo os resultados de uma verificação e as recomendações. A configuração de e-mail baseia-se no sistema operacional que envia o e-mail. O e-mail é enviado a partir do computador no qual o servidor de banco de dados está localizado. É possível configurar o programa de e-mail para o perfil atual.

Para configurar a notificação de e-mail para o perfil atual:

  1. No menu OAT, clique em Health Center > Health Advisor.
  2. Opcional: Para alterar o perfil atual, na página Perfil, clique no perfil na lista e, em seguida, clique em Carregar.
  3. Clique na guia Notificação. Para obter exemplos da configuração, clique no botão de ajuda para a página.
