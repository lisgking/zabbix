Informix-Plug-in 'TimeSeries' für OAT

Das IBM® Informix®-Plug-in 'TimeSeries' für OpenAdmin Tool (OAT) stellt eine grafische Oberfläche zum Verwalten von Datenbankobjekten bereit, die zu einer Zeitreihe gehören, einschließlich TimeSeries-Subtypen, -Containern und -Kalendern.

Voraussetzung:
Für Informix 11.50 und frühere Versionen muss IBM Informix TimeSeries DataBlade® Module installiert und in der Datenbank registriert sein.

Eine Zeitreihe ist eine Datengruppe, die erfasst wird, während sie sich im Laufe der Zeit ändert. Jeder Eintrag erhält eine Zeitmarke und wird in der Datenbank gespeichert. Die Informix-Erweiterung TimeSeries stellt den TimeSeries-Datentyp, die TimeSeries-Container und die TimeSeries-Kalender für die Verwaltung von Zeitreihendaten bereit.

Mit dem Plug-in 'TimeSeries' können Sie die Datenbankobjekte überwachen, die zu Ihrer Zeitreihe gehören:

  • Zeigen Sie die TimeSeries-Subtypen, -Container und -Kalender an, die für die Zeitreihendaten in einer Datenbank verwendet werden.
  • Zeigen Sie die Tabellen und Indizes an, die TimeSeries-Subtypen enthalten.
  • Zeigen Sie die virtuellen Tabellen an, die auf Tabellen basieren, die TimeSeries-Subtypen enthalten.
  • Überwachen Sie den Prozentsatz des Speicherbereichs, der in den Containern und in den Datenbankbereichen für die Container verwendet wird.

Sie können darüber hinaus Container, Kalender und virtuelle Tabellen erstellen und löschen.

Überprüfen von Zeitreihen

Überprüfen Sie Datenbankobjekte, die zu einer Zeitreihe gehören, einschließlich Tabellen, Indizes und virtuellen Tabellen sowie TimeSeries-Subtypen, -Containern und -Kalendern.

Gehen Sie wie folgt vor, um eine Zeitreihe zu überprüfen:

Klicken Sie im Menü von IBM OpenAdmin Tool (OAT) for Informix auf SQL ToolBox > TimeSeries. Es wird eine Liste der Datenbanken und Tabellen auf dem Datenbankserver angezeigt, die Zeitreihendaten enthalten.

  • Wenn Sie Informationen zu einer Datenbank oder Tabelle anzeigen wollen, klicken Sie in der Liste auf das entsprechende Objekt.
  • Wenn Sie weitere Informationen zu einem bestimmten Datenbank- oder Tabellenobjekt anzeigen wollen, klicken Sie in der Objektanzeige auf das Symbol zum Erweitern.

Erstellen eines Containers

Erstellen Sie einen Container für eine Zeitreihe. Sie erstellen mehrere Container, indem Sie mehrere Container pro Datenbankbereich angeben oder mehrere Datenbankbereiche für die Container auswählen.

Gehen Sie wie folgt vor, um einen Container zu erstellen:

  1. Klicken Sie im Menü von IBM OpenAdmin Tool (OAT) for Informix auf SQL ToolBox > TimeSeries.
  2. Klicken Sie auf die Datenbank, die die Zeitreihe enthält.
  3. Klicken Sie im Menü Aktionen auf Container erstellen.
  4. Füllen Sie die Felder auf der Seite aus.

Erstellen eines Kalenders

Erstellen Sie einen Kalender, der einer Zeitreihe zugeordnet wird.

Jede Zeitreihe ist einem Kalender zugeordnet. Ein Kalender definiert eine Reihe gültiger Zeiten, zu denen die Zeitreihe Daten erfassen kann; der Kalender bestimmt, wann und wie oft Einträge akzeptiert werden. Ein Kalender besteht aus einem Startdatum, einem Muster und einem Intervall. Sehen Sie sich das folgende Beispiel an: Das Startdatum des Musters ist ein Sonntag; das Muster lautet 1 aus, 5 ein, 1 aus; das Intervall ist ein Tag. Bei diesem Muster werden am ersten Tag, Sonntag, keine Informationen erfasst. Die Informationen werden an den nächsten fünf Tagen, Montag bis Freitag, täglich erfasst. Am letzten Tag, Samstag, werden keine Informationen erfasst.

Gehen Sie wie folgt vor, um einen Kalender zu erstellen:

  1. Klicken Sie im Menü von IBM OpenAdmin Tool (OAT) for Informix auf SQL ToolBox > TimeSeries.
  2. Klicken Sie auf die Datenbank, die die Zeitreihe enthält.
  3. Klicken Sie im Menü Aktionen auf Kalender erstellen.
  4. Füllen Sie die Felder auf der Seite aus.

Erstellen einer virtuellen Tabelle

Erstellen Sie eine virtuelle Tabelle, die auf einer Tabelle basiert, die einen Subtyp TimeSeries enthält.

Eine virtuelle Tabelle ist eine relationale Tabelle mit demselben Schema wie die Basistabelle, mit Ausnahme der Spalte TimeSeries. Die Spalte TimeSeries wird durch die Spalten des Subtyps TimeSeries ersetzt. Die Elemente in der TimeSeries-Spalte sind einzelne Spalten in der virtuellen Tabelle. Die virtuelle Tabelle enthält alle Daten, die sich in der Basistabelle befinden, einschließlich der Daten, die im Subtyp TimeSeries eingebunden sind.

Gehen Sie wie folgt vor, um eine virtuelle Tabelle zu erstellen:

  1. Klicken Sie im Menü von IBM OpenAdmin Tool (OAT) for Informix auf SQL ToolBox > TimeSeries.
  2. Erweitern Sie in der Liste die Datenbank, die die Basistabelle enthält, und klicken Sie auf die Tabelle.
  3. Klicken Sie im Menü Aktionen auf Virtuelle Tabelle erstellen.
  4. Füllen Sie die Felder auf der Seite aus.
