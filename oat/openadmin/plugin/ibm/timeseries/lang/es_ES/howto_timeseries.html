Plug-in de Informix TimeSeries para OAT

El plug-in IBM® Informix® TimeSeries para la herramienta OpenAdmin (OAT) proporciona una interfaz gráfica para los objetos de bases de datos de administración que están relacionado con una serie temporal, incluidos los subtipos, contenedores y calendarios de TimeSeries.

Requisito previo:
Para Informix 11.50 y anterior, IBM Informix TimeSeries DataBlade® Module debe estar instalado y registrado en la base de datos.

Una serie temporal es un conjunto de datos que se registran a medida que varían. Cada entrada tiene una indicación de fecha y hora y se almacena en la base de datos. La ampliación de Informix TimeSeries ofrece calendarios, contenedores y el tipo de datos TimeSeries para gestionar datos de series temporales.

Con el plug-in de TimeSeries, puede supervisar los objetos de bases de datos relacionados con sus series temporales:

  • Ver los subtipos TimeSeries, los contenedores y los calendarios que se utilizan para los datos de series temporales en una base de datos.
  • Ver las tablas e índices que contienen subtipos TimeSeries.
  • Ver las tablas virtuales que se basan en tablas que contienen subtipos TimeSeries.
  • Supervisar el porcentaje de espacio utilizado en los contenedores y en los espacios de base de datos para los contenedores.

También puede crear y descartar contenedores, calendarios y tablas virtuales.

Revisión de las series temporales

Revise los objetos de base de datos relacionados con una serie temporal, incluidas las tablas, los índices y las tablas virtuales, así como los subtipos TimeSeries, los contenedores y los calendarios.

Para revisar las series temporales:

En el menú de la herramienta IBM OpenAdmin (OAT) para Informix, pulse SQL ToolBox > TimeSeries. Se visualiza una lista de bases de datos y tablas del servidor de bases de datos que contienen datos de series temporales.

  • Para ver información sobre una base de datos o una tabla, pulse en ella en la lista.
  • Para ver más información sobre un objeto de base de datos o de tabla específico, pulse el icono de expansión en el panel del objeto.

Creación de un contenedor

Cree un contenedor para una serie temporal. Puede crear varios contenedores especificando más de un contenedor por espacio de base de datos o seleccionando más de un espacio de base de datos para los contenedores.

Para crear un contenedor:

  1. En el menú de la herramienta IBM OpenAdmin (OAT) para Informix, pulse SQL ToolBox > TimeSeries.
  2. Pulse la base de datos que contiene la serie temporal.
  3. En el menú Acciones, pulse Crear contenedor.
  4. Complete los campos de la página.

Creación de un calendario

Cree un calendario para asociarlo a una serie temporal.

Cada serie temporal está asociada a un calendario. Un calendario define un conjunto de tiempos válidos en los que las series temporales pueden registrar datos; determina cuándo y con qué frecuencia se aceptan entradas. Un calendario consiste en una fecha de inicio, un patrón y un intervalo. Observe este ejemplo: la fecha de inicio del patrón es un domingo; el patrón es 1 desactivado, 5 activado, 1 desactivado; y el intervalo es un día. Para este patrón, no se recopila información en el primer día, domingo. Se recopila información durante los cinco días siguientes, de lunes a viernes. No se recopila información el último día, sábado.

Para crear un calendario:

  1. En el menú de la herramienta IBM OpenAdmin (OAT) para Informix, pulse SQL ToolBox > TimeSeries.
  2. Pulse la base de datos que contiene la serie temporal.
  3. En el menú Acciones, pulse Crear calendario.
  4. Complete los campos de la página.

Creación de una tabla virtual

Cree una tabla virtual basada en una tabla que contiene un subtipo TimeSeries.

Una tabla virtual es una tabla relacional con el mismo esquema que la tabla base, excepto por la columna TimeSeries. La columna TimeSeries se sustituye por las columnas del subtipo TimeSeries. Los elementos de la columna TimeSeries son columnas individuales en la tabla virtual. La tabla virtual contiene todos los datos que hay en la tabla base, incluidos los datos encapsulados en el subtipo TimeSeries.

Para crear una tabla virtual:

  1. En el menú de la herramienta IBM OpenAdmin (OAT) para Informix, pulse SQL ToolBox > TimeSeries.
  2. En la lista, expanda la base de datos que contiene la tabla base y pulse la tabla.
  3. En el menú Acciones, pulse Crear tabla virtual.
  4. Complete los campos de la página.
