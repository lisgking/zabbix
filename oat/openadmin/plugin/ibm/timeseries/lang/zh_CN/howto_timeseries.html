Informix TimeSeries Plug-in for OAT

IBM® Informix® TimeSeries Plug-in for OpenAdmin Tool (OAT) 提供图形界面，用于管理与时间系列相关的数据库对象，包括 TimeSeries 子类型、容器和日历。

先决条件：
对于 Informix 11.50 和更早版本，必须在数据库中安装并注册 IBM Informix TimeSeries DataBlade® 模块。

时间系列是因随时间发生变化而被记录的一组数据。每个条目都具有时间戳记，并存储在数据库中。Informix TimeSeries 扩展提供了 TimeSeries 数据类型、容器和日历，用于管理时间系列数据。

使用 TimeSeries 插件，您可以监视与时间系列相关的数据库对象：

  • 查看用于数据库中时间系列数据的 TimeSeries 子类型、容器和日历。
  • 查看包含 TimeSeries 子类型的表和索引。
  • 查看基于包含 TimeSeries 子类型的表的虚拟表。
  • 监视在容器和容器的数据库空间中使用的空间百分比。

您还可以创建和删除容器、日历和虚拟表。

复审时间系列

复审与时间系列相关的数据库对象，包括表、索引、虚拟表、TimeSeries 子类型、容器和日历。

要复审时间系列：

在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > TimeSeries。将显示数据库服务器中包含时间系列数据的数据库和表的列表。

  • 要查看有关数据库或表的信息，请在列表中单击该项。
  • 要查看有关特定数据库对象或表对象的更多信息，请单击对象面板上的展开图标。

创建容器

为时间系列创建容器。通过为每个数据库空间指定多个容器或通过为容器选择多个数据库空间，创建多个容器。

要创建容器：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > TimeSeries。
  2. 单击包含时间系列的数据库。
  3. 在操作菜单上，单击创建容器。
  4. 填写页面上的字段。

创建日历

创建日历以与时间系列关联。

每个时间系列都与日历关联。日历定义了一组有效的时间，时间系列可在这些时间记录数据；日历将确定接受条目的时间和频率。日历由开始日期、模式和时间间隔组成。看一下这个示例：模式开始日期为星期日；模式为 1 禁用，5 启用，1 禁用；时间间隔为一天。对于此模式，未收集第一天（即星期日）的信息。对接下来五天（周一到周五）每一天的信息都进行了收集。最后一天（即星期六）的信息未收集。

要创建日历：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > TimeSeries。
  2. 单击包含时间系列的数据库。
  3. 在操作菜单上，单击创建日历。
  4. 填写页面上的字段。

创建虚拟表

创建以包含 TimeSeries 子类型的表为基础的虚拟表。

虚拟表是与基本表具有相同模式（除了 TimeSeries 列）的关系表。TimeSeries 列替换为 TimeSeries 子类型的列。TimeSeries 列中的元素是虚拟表中的单个列。虚拟表包含基本表中的所有数据，包括 TimeSeries 子类型中封装的数据。

要创建虚拟表：

  1. 在 IBM OpenAdmin Tool (OAT) for Informix 的菜单上，单击 SQL 工具箱 > TimeSeries。
  2. 在列表中，展开包含基本表的数据库并单击该表。
  3. 在操作菜单上，单击创建虚拟表。
  4. 填写页面上的字段。
