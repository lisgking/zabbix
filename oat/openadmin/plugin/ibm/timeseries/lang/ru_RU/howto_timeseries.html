Подключаемый модуль репликации Informix для OAT

IBM® Informix® TimeSeries Plug-in for OpenAdmin Tool (OAT) обеспечивает графический интерфейс для управления объектами баз данных, связанных с временными рядами, включая подтипы TimeSeries, контейнеры и календари.

Предварительное требование:
Для Informix 11.50 и более ранних в базе данных должен быть установлен и зарегистрирован модуль IBM Informix TimeSeries DataBlade®.

Данные TimeSeries - это набор записанных данных, значения которых меняются со временем. Каждая запись содержит отметку времени и сохраняется в базе данных. Расширение Informix TimeSeries поддерживает тип данных TimeSeries, контейнеры и календари для управлениями данными временных рядов.

При помощи подключаемого модуля TimeSeries можно выполнять мониторинг объектов баз данных, связанных с данными TimeSeries:

  • Просматривать подтипы TimeSeries, контейнеры и календари, используемые для данных TimeSeries в базе данных.
  • Просматривать таблицы и индексы, содержащие подтипы TimeSeries.
  • Просматривать виртуальные таблицы, основанных на стандартных таблицах с подтипами TimeSeries.
  • Отслеживать процент используемого пространства в контейнерах и пространствах баз данных для контейнеров.

Можно также создавать и отбрасывать контейнеры, календари и виртуальные таблицы.

Просмотр данных TimeSeries

Просмотр объектов баз данных, связанные с данными TimeSeries, включая таблицы, индексы и виртуальные таблицы, а также подтипы, контейнеры и календари TimeSeries.

Чтобы просмотреть данные TimeSeries:

В меню IBM OpenAdmin Tool (OAT) for Informix выберите SQL ToolBox > TimeSeries. Выводится список баз данных и таблиц на сервере баз данных, содержащих данные TimeSeries.

  • Для просмотра информации о базе данных или таблице щелкните по ней в списке.
  • Для просмотра дополнительной информации об определенном объекте базы данных или объекте таблицы нажмите кнопку раскрытия на панели объектов.

Создание контейнера

Создание контейнера для данных TimeSeries. Создайте несколько контейнеров, задав несколько контейнеров на пространство баз данных или несколько пространств баз данных для контейнеров.

Чтобы создать контейнер:

  1. В меню IBM OpenAdmin Tool (OAT) for Informix выберите SQL ToolBox > TimeSeries.
  2. Щелкните по базе данных, которая содержит данные TimeSeries.
  3. В меню Действия выберите пункт Создать контейнер.
  4. Заполните поля на этой странице.

Создание календаря

Создайте календарь, связанный с данными TimeSeries.

Каждый временной ряд связан с календарем. Календарь определяет набор допустимых моментов времени, когда могут записываться данные TimeSeries; он определяет, когда и как часто выполняется прием данных. Календарь содержит начальную дату, шаблон и интервал. Рассмотрим такой пример: начальная дата шаблона приходится на воскресенье. Шаблон имеет вид 1 off (нет), 5 on (да), 1 off (нет), а интервал - 1 сутки. Для такого шаблона информация не записывается в первый день - воскресенье. Каждый из следующих пяти дней, с понедельника по пятницу, выполняется сбор информации. Последний день, в субботу, информация не собирается.

Чтобы создать календарь:

  1. В меню IBM OpenAdmin Tool (OAT) for Informix выберите SQL ToolBox > TimeSeries.
  2. Щелкните по базе данных, которая содержит данные TimeSeries.
  3. В меню Действия выберите пункт Создать календарь.
  4. Заполните поля на этой странице.

Создание виртуальной таблицы

Создайте виртуальную таблицу, основанную на стандартной таблице с подтипом TimeSeries.

Виртуальная таблица - это реляционная таблица с той же схемой, что и базовая таблица, за исключением столбца TimeSeries. Столбец TimeSeries заменяется на столбцы подтипа TimeSeries. Элементы столбца TimeSeries - это отдельные столбцы в виртуальной таблице. Виртуальная таблица содержит все данные, которые есть в базовой таблице, включая данные, инкапсулированные в подтипе TimeSeries.

Чтобы создать виртуальную таблицу:

  1. В меню IBM OpenAdmin Tool (OAT) for Informix выберите SQL ToolBox > TimeSeries.
  2. Раскройте в списке базу данных, содержащую базовую таблицу, и щелкните по этой таблице.
  3. В меню Действия выберите пункт Создать виртуальную таблицу.
  4. Заполните поля на этой странице.
