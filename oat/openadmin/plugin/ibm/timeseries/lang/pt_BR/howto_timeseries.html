Informix TimeSeries Plug-in for OAT

O IBM® Informix® TimeSeries Plug-in for OpenAdmin Tool (OAT) fornece uma interface gráfica para administrar objetos de banco de dados que estão relacionados a uma série temporal, incluindo subtipos, contêineres e calendários do TimeSeries.

Pré-requisito:
Para Informix 11.50 e anterior, o IBM Informix TimeSeries DataBlade® Module deve ser instalado e registrado no banco de dados.

Uma série temporal é um conjunto de dados que é registrado à medida que varia com o passar do tempo. Cada entrada possui um registro de data e hora e é armazenada no banco de dados. A extensão do Informix TimeSeries fornece o tipo de dados, contêineres e calendários do TimeSeries para gerenciar dados de série temporal.

Com o plug-in TimeSeries, é possível monitorar os objetos de banco de dados relacionados à série temporal:

  • Visualize os subtipos de TimeSeries, contêineres e calendários que são usados para os dados de série temporal em um banco de dados.
  • Visualize as tabelas e índices que contêm os subtipos do TimeSeries.
  • Visualize as tabelas virtuais que se baseiam em tabelas contendo subtipos do TimeSeries.
  • Monitore a porcentagem do espaço usado nos contêineres e nos dbspaces dos contêineres.

É possível também criar e eliminar contêineres, calendários e tabelas virtuais.

Revisando a Série Temporal

Revise os objetos de banco de dados relacionados a uma série temporal, incluindo tabelas, índices e tabelas virtuais, e subtipos, contêineres e calendários do TimeSeries.

Para revisar a série temporal:

No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferrametnas de SQL > TimeSeries. É exibida uma lista dos bancos de dados e tabelas no servidor de banco de dados que contém dados de série temporal.

  • Para visualizar informações sobre um banco de dados ou uma tabela, clique nela na lista.
  • Para visualizar informações adicionais sobre um objeto de banco de dados específico ou objeto de tabela, clique no ícone de expansão no painel do objeto.

Criando um Contêiner

Crie um contêiner para uma série temporal. Crie diversos contêineres especificando mais de um contêiner por dbspace ou selecionando mais de um dbspace para os contêineres.

Para criar um contêiner:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferrametnas de SQL > TimeSeries.
  2. Clique no banco de dados que contém a série temporal.
  3. No menu Ações, clique em Criar Contêiner.
  4. Preencha os campos na página.

Criando um Calendário

Crie um calendário com uma série temporal.

Cada série temporal é associada a um calendário. Um calendário define um conjunto de horários válidos no qual a série temporal pode registrar dados; ele determina quando e com que frequência as entradas são aceitas. Um calendário consiste em uma data de início, um padrão e um intervalo. Considere este exemplo: a data de início padrão é um domingo; o padrão é 1 desligado, 5 ligado, 1 desligado; e o intervalo é um dia. Para esse padrão, as informações não são reunidas no primeiro dia, domingo. As informações são reunidas todos os dias nos próximos cinco dias, segunda a sexta. As informações não são reunidas no último dia, sábado.

Para criar um calendário:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferrametnas de SQL > TimeSeries.
  2. Clique no banco de dados que contém a série temporal.
  3. No menu Ações, clique em Criar Calendário.
  4. Preencha os campos na página.

Criando uma Tabela Virtual

Crie uma tabela virtual baseada em uma tabela que contenha um subtipo do TimeSeries.

Uma tabela virtual é uma tabela relacional com o mesmo esquema que a tabela base, exceto para a coluna TimeSeries. A coluna do TimeSeries é substituída pelas colunas do subtipo do TimeSeries. Os elementos na coluna do TimeSeries são individuais na tabela virtual. A tabela virtual contém todos os dados que estão na tabela base, incluindo os dados contidos no subtipo TimeSeries.

Para criar uma tabela virtual:

  1. No menu do IBM OpenAdmin Tool (OAT) for Informix, clique em Caixa de Ferrametnas de SQL > TimeSeries.
  2. Na lista, expanda o banco de dados que contém a tabela base e clique na tabela.
  3. No menu Ações, clique em Criar Tabela Virtual.
  4. Preencha os campos na página.
